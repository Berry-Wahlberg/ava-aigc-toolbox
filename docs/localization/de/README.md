# AVA AIGC Toolbox Benutzerdokumentation

> **Die englische Version ist maßgeblich**

Willkommen zur umfassenden Benutzerdokumentation für das AVA AIGC Toolbox! Diese Dokumentation führt Sie durch alle Aspekte der Nutzung dieser leistungsstarken plattformübergreifenden AI-generierten Bildverwaltung-Anwendung.

## Einführung

AVA AIGC Toolbox ist eine funktionsreiche Anwendung, die Ihnen hilft, Ihre wachsende Sammlung von KI-generierten Bildern zu verwalten, zu organisieren und zu bearbeiten. Mit Avalonia und .NET entwickelt, bietet sie ein einheitliches Erlebnis unter Windows, macOS und Linux-Betriebssystemen.

## Dokumentationsübersicht

Diese Dokumentation ist so organisiert, dass Sie sich schnell zurechtfinden und dann fortgeschrittene Funktionen erkunden können, sobald Sie mit der Anwendung vertraut sind.

## Systemanforderungen

| Kategorie | Mindestanforderungen | Empfohlen |
|----------|---------------------|-------------|
| **Betriebssystem** | Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+, Fedora 32+) | Windows 11, macOS 12+, neueste Linux-Distribution |
| **.NET Runtime** | .NET 7.0 oder höher | .NET 7.0 oder höher |
| **Festplattenspeicher** | 100 MB freier Speicher | 500 MB oder mehr |
| **RAM** | 2 GB | 4 GB oder mehr |
| **Prozessor** | Dual-Core-CPU | Quad-Core-CPU oder besser |
| **Anzeige** | 1024x768 Auflösung | 1920x1080 oder höher |
