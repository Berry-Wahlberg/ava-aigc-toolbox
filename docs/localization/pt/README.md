# Documentação do Usuário da AVA AIGC Toolbox

> **A versão em português é a oficial**

Bem-vindo à documentação completa do usuário da AVA AIGC Toolbox! Esta documentação irá guiá-lo através de todos os aspectos do uso deste poderoso aplicativo de gerenciamento de imagens geradas por IA multiplataforma.

## Introdução

A AVA AIGC Toolbox é um aplicativo rico em recursos projetado para ajudá-lo a gerenciar, organizar e manipular sua crescente coleção de imagens geradas por IA. Construído com Avalonia e .NET, ele oferece uma experiência unificada em sistemas operacionais Windows, macOS e Linux.

## Visão Geral da Documentação

Esta documentação está organizada para ajudá-lo a começar rapidamente e, em seguida, explorar recursos mais avançados à medida que se familiariza com o aplicativo.

## Requisitos do Sistema

| Categoria | Requisitos Mínimos | Recomendado |
|----------|---------------------|-------------|
| **Sistema Operacional** | Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+, Fedora 32+) | Windows 11, macOS 12+, última distribuição Linux |
| **Runtime .NET** | .NET 7.0 ou posterior | .NET 7.0 ou posterior |
| **Espaço em Disco** | 100 MB de espaço livre | 500 MB ou mais |
| **RAM** | 2 GB | 4 GB ou mais |
| **Processador** | CPU dual-core | CPU quad-core ou melhor |
| **Tela** | Resolução 1024x768 | 1920x1080 ou superior |

## Formatos de Imagem Suportados

- JPEG/JPG
- PNG
- WebP
- TIFF
- BMP
- GIF (apenas estático)
