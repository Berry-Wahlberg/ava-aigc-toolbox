> **A versão em português é a oficial**

## Personalizando Atalhos do Teclado

Você pode personalizar os atalhos do teclado para se adequar ao seu fluxo de trabalho:

#### Passos para Personalizar Atalhos do Teclado:
1. Vá para `Arquivo > Configurações > Atalhos do Teclado`
2. Nas configurações de atalhos do teclado, você pode:
   - **Visualizar** todos os atalhos disponíveis
   - **Pesquisar** atalhos específicos
   - **Editar** atalhos existentes
   - **Redefinir** atalhos para os padrões
3. Para editar um atalho:
   - Clique no campo do atalho ao lado da ação
   - Pressione a nova combinação de teclas
   - Clique em **Salvar** para aplicar o novo atalho
4. Clique em **Salvar** para salvar todas as alterações
5. Reinicie o aplicativo para que as alterações tenham efeito

## Importando/Exportando Configurações de Atalhos

#### Passos para Exportar Configuração de Atalhos:
1. Vá para `Arquivo > Configurações > Atalhos do Teclado`
2. Clique em **Exportar Configuração**
3. Escolha a pasta de destino para o arquivo de exportação
4. Digite um nome para o arquivo
5. Clique em **Salvar** para exportar a configuração

#### Passos para Importar Configuração de Atalhos:
1. Vá para `Arquivo > Configurações > Atalhos do Teclado`
2. Clique em **Importar Configuração**
3. Selecione o arquivo de configuração do seu sistema de arquivos
4. Clique em **Abrir** para importar a configuração
5. Clique em **Salvar** para aplicar os atalhos importados
6. Reinicie o aplicativo para que as alterações tenham efeito

## Melhores Práticas para Atalhos do Teclado

1. **Aprenda Atalhos Comuns**: Comece com os atalhos mais comumente usados
2. **Personalize para o Seu Fluxo de Trabalho**: Ajuste os atalhos para corresponder ao seu fluxo de trabalho preferido
3. **Evite Conflitos**: Certifique-se de que os novos atalhos não entrem em conflito com os existentes
4. **Documente Seus Atalhos**: Mantenha um registro dos seus atalhos personalizados
5. **Praticar Regularmente**: Use atalhos com frequência para desenvolver memória muscular
6. **Redefina se Necessário**: Se você ficar confuso, redefina os atalhos para os padrões
