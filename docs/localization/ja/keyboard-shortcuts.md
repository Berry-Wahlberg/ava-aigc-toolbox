# キーボードショートカット

> **日本語版が公式バージョンです**

AVA AIGC ツールボックスは、効率的に作業するための包括的なキーボードショートカットセットを提供します。このガイドでは、利用可能なすべてのキーボードショートカットとそのカスタマイズ方法を一覧表示します。

## 一般的なショートカット

| ショートカット | アクション |
|----------------|------------|
| `Ctrl/Cmd + O` | ライブラリを開く |
| `Ctrl/Cmd + S` | 現在の変更を保存 |
| `Ctrl/Cmd + Q` | アプリケーションを終了 |
| `Ctrl/Cmd + ,` | 設定を開く |
| `Ctrl/Cmd + H` | サイドバーを表示/非表示 |
| `Ctrl/Cmd + L` | 検索バーにフォーカス |
| `Ctrl/Cmd + F` | 現在のビュー内で検索 |
| `F1` | ドキュメントを開く |
| `F12` | デバッグモードを切り替え |

## キーボードショートカットのカスタマイズ

ワークフローに合わせてキーボードショートカットをカスタマイズできます：

#### キーボードショートカットをカスタマイズする手順：
1. `ファイル > 設定 > キーボードショートカット` に移動します
2. キーボードショートカット設定では、以下を行えます：
   - **表示**：すべての利用可能なショートカットを表示
   - **検索**：特定のショートカットを検索
   - **編集**：既存のショートカットを編集
   - **リセット**：ショートカットをデフォルトにリセット
3. ショートカットを編集するには：
   - アクションの横にあるショートカットフィールドをクリックします
   - 新しいキーの組み合わせを押します
   - **保存**をクリックして新しいショートカットを適用します
4. **保存**をクリックしてすべての変更を保存します
5. 変更を有効にするためにアプリケーションを再起動します

## ショートカット設定のインポート/エクスポート

#### ショートカット設定をエクスポートする手順：
1. `ファイル > 設定 > キーボードショートカット` に移動します
2. **設定をエクスポート**をクリックします
3. エクスポートファイルの保存先フォルダを選択します
4. ファイル名を入力します
5. **保存**をクリックして設定をエクスポートします

#### ショートカット設定をインポートする手順：
1. `ファイル > 設定 > キーボードショートカット` に移動します
2. **設定をインポート**をクリックします
3. ファイルシステムから設定ファイルを選択します
4. **開く**をクリックして設定をインポートします
5. **保存**をクリックしてインポートしたショートカットを適用します
6. 変更を有効にするためにアプリケーションを再起動します

## キーボードショートカットのベストプラクティス

1. **一般的なショートカットを学ぶ**：最もよく使用されるショートカットから始めます
2. **ワークフローに合わせてカスタマイズ**：好みのワークフローに合わせてショートカットを調整します
3. **競合を避ける**：新しいショートカットが既存のものと競合しないようにします
4. **ショートカットを文書化**：カスタムショートカットの記録を保持します
5. **定期的に練習**：筋肉の記憶を養うために頻繁にショートカットを使用します
6. **必要に応じてリセット**：混乱した場合は、ショートカットをデフォルトにリセットします

## キーボードショートカットのトラブルシューティング

### ショートカットが機能しない
- **フォーカスを確認**：正しいビューにフォーカスがあることを確認します
- **競合を確認**：他のアプリケーションがショートカットを妨害していないことを確認します
- **設定を確認**：ショートカットが設定で有効になっていることを確認します
- **アプリケーションを再起動**：アプリケーションを再起動してみます
- **ショートカットをリセット**：問題が続く場合は、ショートカットをデフォルトにリセットします

### ショートカットの競合
- **競合するアプリケーションを特定**：どのアプリケーションが競合しているかを特定します
- **ショートカットを変更**：設定でショートカットを変更します
- **競合を無効にする**：他のアプリケーションで競合するショートカットを無効にします

### カスタムショートカットが保存されない
- **アクセス権を確認**：設定ファイルへの書き込み権限があることを確認します
- **アプリケーションを再起動**：変更を適用するためにアプリケーションを再起動します
- **リセットして再設定**：ショートカットをデフォルトにリセットしてから再設定します
