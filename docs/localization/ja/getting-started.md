# はじめに

## AVA AIGC ツールボックスへようこそ

AVA AIGC ツールボックスを選択していただきありがとうございます！このガイドは、アプリケーションの基本操作と主な機能の探索をサポートします。

## 初回起動

### 1. アプリケーションの起動
- **Windows**: デスクトップショートカットをダブルクリックするか、スタートメニューから起動します
- **macOS**: アプリケーションフォルダから開くか、Spotlightを使用します
- **Linux**: アプリケーションメニューから起動するか、ターミナルで `ava-aigc-toolbox` を実行します

### 2. 初期設定

アプリケーションを初めて起動すると：

1. **ウェルカム画面**
   - 以下のオプションが表示されます：
     - 空のライブラリで開始
     - 既存の画像をインポート
     - アプリケーションの詳細を学ぶ

2. **オプションの選択**
   - **空のライブラリで開始**: 画像用の新しいデータベースを作成します
   - **既存の画像をインポート**: 画像のあるフォルダを選択してインポートできます
   - **詳細を学ぶ**: ドキュメントを開きます

3. **データベースの場所**
   - アプリケーションは自動的にデータベースファイルを作成します：
     - Windows: `%APPDATA%/AIGenManager/aigenmanager.db`
     - macOS: `~/.local/share/AIGenManager/aigenmanager.db`
     - Linux: `~/.local/share/AIGenManager/aigenmanager.db`

## 基本的なナビゲーション

メインウィンドウは複数のセクションに分かれています：

### 1. サイドバー
- **フォルダ**: 画像フォルダをナビゲートします
- **アルバム**: 画像アルバムにアクセスします
- **タグ**: タグで閲覧およびフィルタリングします
- **すべての画像**: ライブラリ内のすべての画像を表示します

### 2. メインコンテンツエリア
- **画像グリッド**: グリッドレイアウトで画像を表示します
- **画像詳細**: 画像が選択されたときにメタデータとプロパティを表示します

### 3. ツールバー
- **インポート**: 新しい画像をライブラリに追加します
- **並べ替え**: 画像の並べ替え順序を変更します
- **フィルター**: 画像グリッドにフィルターを適用します
- **表示**: グリッド表示とリスト表示を切り替えます

### 4. ステータスバー
- 画像の総数を表示します
- 現在のフィルターまたは検索条件を表示します
- アプリケーションのステータスを表示します

## 画像の追加

### 1. 画像のインポート

#### ファイルシステムから
1. ツールバーの**インポート**ボタンをクリックします
2. **フォルダからインポート**を選択します
3. 画像のあるフォルダを選択します
4. **フォルダを選択**をクリックしてインポートを開始します

#### ドラッグアンドドロップ
1. ファイルエクスプローラー/ファインダーを開きます
2. 1つ以上の画像を選択します
3. アプリケーションのメインコンテンツエリアにドラッグします
4. 画像がライブラリに追加されます

### 2. 画像メタデータ

画像がインポートされると、アプリケーションは自動的に以下を抽出します：
- ファイル名とパス
- ファイルサイズと寸法
- 作成日と変更日
- AI生成メタデータ（利用可能な場合）：
  - プロンプト
  - ネガティブプロンプト
  - ステップ数とサンプラー
  - CFGスケールとシード
  - モデル情報

## 画像の整理

### 1. フォルダの使用

- アプリケーションはファイルシステムのフォルダ構造を反映します
- サイドバーでフォルダをナビゲートして、特定の場所の画像を表示します
- ファイルシステムで作成された新しいフォルダは自動的に検出されます

### 2. アルバムの作成

1. サイドバーの「アルバム」の横にある**+**ボタンをクリックします
2. アルバムの名前を入力します
3. Enterキーを押して作成します
4. グリッドからアルバムに画像をドラッグして追加します

### 3. タグの追加

#### 単一画像への追加
1. グリッドで画像を選択します
2. 詳細パネルで「タグ」セクションを見つけます
3. **+**ボタンをクリックします
4. タグ名を入力してEnterキーを押します

#### 複数画像への追加
1. Ctrl/Cmd + クリックで複数の画像を選択します
2. 右クリックして**タグを追加**を選択します
3. カンマで区切ってタグ名を入力します
4. **追加**をクリックして、選択したすべての画像にタグを適用します

## 画像の操作

### 1. 画像の表示

- **単一クリック**: 画像を選択して詳細を表示します
- **ダブルクリック**: フルスクリーンビューで画像を開きます
- **右クリック**: 追加オプションが含まれるコンテキストメニューを開きます

### 2. 画像詳細

画像が選択されると、以下が表示されます：
- 基本情報（ファイル名、サイズ、寸法）
- AIメタデータ（プロンプト、ネガティブプロンプトなど）
- 画像に関連付けられたタグ
- 評価とお気に入りの状態

### 3. メタデータの編集

1. 画像を選択します
2. 詳細パネルで編集可能なフィールドをクリックします
3. 変更を加えます
4. Enterキーを押すか、フィールド外をクリックして保存します

### 4. 評価とお気に入り

- **評価**: 詳細パネルの星をクリックして画像に評価を付けます（1-5つ星）
- **お気に入り**: ハートアイコンをクリックして画像をお気に入りとしてマークします
- **削除予定**: 後で簡単に削除できるように、削除予定として画像にマークします

## 検索とフィルタリング

### 1. 基本検索

1. ウィンドウ上部の検索ボックスに入力します
2. 入力と同時に結果が表示されます
3. 検索はファイル名、タグ、およびメタデータと一致します

### 2. 高度なフィルタリング

1. ツールバーの**フィルター**ボタンをクリックします
2. フィルター条件を設定します：
   - フォルダ
   - アルバム
   - タグ
   - 評価
   - 日付範囲
   - 寸法
   - AIメタデータ（モデル、サンプラーなど）
3. **適用**をクリックしてフィルタリングされた結果を表示します

## 画像のエクスポート

### 1. 単一画像のエクスポート

1. 画像を選択します
2. 右クリックして**画像をエクスポート**を選択します
3. 保存先フォルダを選択します
4. **保存**をクリックします

### 2. 複数画像のエクスポート

1. 複数の画像を選択します
2. 右クリックして**選択した画像をエクスポート**を選択します
3. 保存先フォルダを選択します
4. **保存**をクリックします

### 3. メタデータ付きのエクスポート

- エクスポート時に、メタデータを含めることができます
- エクスポートダイアログで「メタデータを含める」オプションをチェックします
- メタデータはエクスポートされた画像に埋め込まれます
