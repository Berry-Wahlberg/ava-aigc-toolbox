# AVA AIGC ツールボックス ユーザードキュメント

> **英語版が準拠となります**

AVA AIGC ツールボックスの包括的なユーザードキュメントへようこそ！このドキュメントは、この強力なクロスプラットフォーム AI 生成画像管理アプリケーションのあらゆる側面をガイドします。

## 導入

AVA AIGC ツールボックスは、AI 生成画像の成長するコレクションを管理、整理、操作するために設計された機能豊富なアプリケーションです。Avalonia と .NET で構築され、Windows、macOS、Linux オペレーティング システム間で統一されたエクスペリエンスを提供します。

## ドキュメントの概要

このドキュメントは、アプリケーションに慣れるにつれて、迅速に始められるように、そしてより高度な機能を探索できるように組織されています。

## システム要件

| カテゴリ | 最小要件 | 推奨 |
|----------|----------|------|
| **オペレーティングシステム** | Windows 10+、macOS 10.15+、Linux (Ubuntu 20.04+、Fedora 32+) | Windows 11、macOS 12+、最新の Linux ディストリビューション |
| **.NET ランタイム** | .NET 7.0 以降 | .NET 7.0 以降 |
| **ディスクスペース** | 100 MB の空きスペース | 500 MB 以上 |
| **RAM** | 2 GB | 4 GB 以上 |
| **プロセッサ** | デュアルコア CPU | クアッドコア CPU 以上 |
| **ディスプレイ** | 1024x768 解像度 | 1920x1080 以上 |

## サポートされている画像形式

- JPEG/JPG
- PNG
- WebP
- TIFF
- BMP
- GIF（静止画のみ）
