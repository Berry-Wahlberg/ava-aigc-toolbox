# Documentazione Utente di AVA AIGC Toolbox

> **La versione inglese è quella ufficiale**

Benvenuto nella documentazione utente completa per AVA AIGC Toolbox! Questa documentazione ti guiderà attraverso tutti gli aspetti dell'uso di questa potente applicazione multipiattaforma per la gestione di immagini generate da IA.

## Introduzione

AVA AIGC Toolbox è un'applicazione ricca di funzionalità progettata per aiutarti a gestire, organizzare e manipolare la tua crescente collezione di immagini generate da IA. Costruita con Avalonia e .NET, offre un'esperienza unificata su sistemi operativi Windows, macOS e Linux.

## Panoramica della Documentazione

Questa documentazione è organizzata per aiutarti a iniziare rapidamente e poi a esplorare funzionalità più avanzate man mano che ti familiarizzi con l'applicazione.

## Requisiti di Sistema

| Categoria | Requisiti Minimi | Raccomandati |
|----------|---------------------|-------------|
| **Sistema Operativo** | Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+, Fedora 32+) | Windows 11, macOS 12+, ultima distribuzione Linux |
| **Runtime .NET** | .NET 7.0 o successivo | .NET 7.0 o successivo |
| **Spazio su Disco** | 100 MB di spazio libero | 500 MB o più |
| **RAM** | 2 GB | 4 GB o più |
| **Processore** | CPU dual-core | CPU quad-core o superiore |
| **Schermo** | Risoluzione 1024x768 | 1920x1080 o superiore |
