# Руководство по началу работы

> **Русская версия является официальной**

## Добро пожаловать в AVA AIGC Toolbox

Спасибо за выбор AVA AIGC Toolbox! Это руководство поможет вам начать работу с приложением и изучить его ключевые функции.

## Первый запуск

### 1. Запуск приложения
- **Windows**: Дважды кликните по ярлыку на рабочем столе или используйте меню Пуск
- **macOS**: Откройте из папки Applications или используйте Spotlight
- **Linux**: Запустите из меню приложений или выполните `ava-aigc-toolbox` в терминале

### 2. Начальная настройка

Когда вы запускаете приложение впервые:

1. **Экран приветствия**
   - Вы увидите экран приветствия с вариантами:
     - Начать с пустой библиотеки
     - Импортировать существующие изображения
     - Узнать больше о приложении

2. **Выберите вариант**
   - **Начать с пустой библиотеки**: Создает новую базу данных для ваших изображений
   - **Импортировать существующие изображения**: Позволяет выбрать папки для импорта изображений
   - **Узнать больше**: Открывает документацию

3. **Расположение базы данных**
   - Приложение автоматически создает файл базы данных:
     - Windows: `%APPDATA%/AIGenManager/aigenmanager.db`
     - macOS: `~/.local/share/AIGenManager/aigenmanager.db`
     - Linux: `~/.local/share/AIGenManager/aigenmanager.db`

## Основная навигация

Главное окно разделено на несколько секций:

### 1. боковая панель
- **Папки**: Навигация по папкам с изображениями
- **Альбомы**: Доступ к альбомам изображений
- **Теги**: Просмотр и фильтрация по тегам
- **Все изображения**: Просмотр всех изображений в библиотеке

### 2. Основная область контента
- **Сетка изображений**: Отображение изображений в сеточном макете
- **Детали изображения**: Показ метаданных и свойств при выборе изображения

### 3. Панель инструментов
- **Импорт**: Добавление новых изображений в библиотеку
- **Сортировка**: Изменение порядка сортировки изображений
- **Фильтр**: Применение фильтров к сетке изображений
- **Просмотр**: Переключение между сеточным и списковым просмотром

### 4. Строка состояния
- Показывает общее количество изображений
- Отображает текущие критерии фильтрации или поиска
- Показывает состояние приложения

## Добавление изображений

### 1. Импорт изображений

#### Из файловой системы
1. Нажмите кнопку **Импорт** на панели инструментов или перейдите в `Файл > Импорт > Импорт изображений`
2. В диалоговом окне импорта нажмите **Добавить папку** и выберите одну или несколько папок, содержащих изображения
3. Настройте параметры импорта:
   - **Включить подпапки**: Импортировать изображения из всех поддиректорий
   - **Перезаписать существующие изображения**: Заменять изображения с тем же путем в библиотеке
   - **Извлекать метаданные**: Автоматически извлекать метаданные ИИ из изображений
   - **Генерировать эскизы**: Создавать или перегенерировать эскизы для всех импортированных изображений
4. Нажмите **Начать импорт** для начала процесса импорта
5. Мониторьте прогресс в диалоговом окне импорта
6. Нажмите **Завершить** когда импорт завершен

#### Перетащите и отпустите

Вы также можете импортировать изображения, перетащив их в приложение:

#### Шаги для импорта через перетаскивание:
1. Откройте менеджер файлов системы (Explorer/Finder)
2. Выберите одно или несколько изображений или папок
3. Перетащите их в окно AVA AIGC Toolbox
4. Появится диалоговое окно подтверждения с параметрами импорта
5. Настройте параметры по желанию
6. Нажмите **Импортировать** для начала процесса импорта

### 2. Метаданные изображения

При импорте изображений приложение автоматически извлекает:
- Имя файла и путь
- Размер и размеры файла
- Даты создания и изменения
- Метаданные, сгенерированные ИИ (если доступны):
  - Промпт
  - Негативный промпт
  - Шаги и сэмплер
  - Масштаб CFG и сид
  - Информация о модели

## Организация изображений

### 1. Использование папок

- Приложение отражает структуру папок вашей файловой системы
- Навигация по папкам в боковой панели для просмотра изображений в конкретных местах
- Новые папки, созданные в файловой системе, автоматически обнаруживаются

### 2. Создание альбомов

1. Нажмите кнопку **+** рядом с "Альбомы" в боковой панели
2. Введите имя для альбома
3. Нажмите Enter для создания
4. Перетащите изображения из сетки в альбом, чтобы добавить их

### 3. Добавление тегов

#### К одному изображению
1. Выберите изображение в сетке
2. В панели деталей найдите раздел "Теги"
3. Нажмите кнопку **+**
4. Введите имя тега и нажмите Enter

#### К нескольким изображениям
1. Выберите несколько изображений, используя Ctrl/Cmd + клик
2. Щелкните правой кнопкой мыши и выберите **Добавить теги**
3. Введите имена тегов, разделенные запятыми
4. Нажмите **Добавить** для применения тегов ко всем выбранным изображениям

## Работа с изображениями

### 1. Просмотр изображений

- **Одиночный клик**: Выберите изображение, чтобы увидеть детали
- **Двойной клик**: Откройте изображение в полноэкранном режиме
- **Щелчок правой кнопкой мыши**: Откройте контекстное меню с дополнительными опциями

### 2. Детали изображения

Когда изображение выбрано, вы увидите:
- Основная информация (имя файла, размер, размеры)
- Метаданные ИИ (промпт, негативный промпт и т.д.)
- Теги, связанные с изображением
- Рейтинг и статус избранного

### 3. Редактирование метаданных

1. Выберите изображение
2. В панели деталей кликните по любому редактируемому полю
3. Внесите изменения
4. Нажмите Enter или кликните вне поля, чтобы сохранить

### 4. Рейтинг и избранное

- **Рейтинг**: Нажмите на звезды в панели деталей, чтобы оценить изображение (1-5 звезд)
- **Избранное**: Нажмите на иконку сердца, чтобы отметить изображение как избранное
- **Для удаления**: Отметьте изображения для удаления, чтобы легко удалить их позже

## Поиск и фильтрация

### 1. Базовый поиск

1. Введите текст в поисковую строку в верхней части окна
2. Результаты появятся автоматически по мере ввода
3. Поиск совпадает с именами файлов, тегами и метаданными

### 2. Расширенная фильтрация

1. Нажмите кнопку **Фильтр** на панели инструментов
2. Задайте критерии фильтрации:
   - Папка
   - Альбом
   - Теги
   - Рейтинг
   - Диапазон дат
   - Размеры
   - Метаданные ИИ (модель, сэмплер и т.д.)
3. Нажмите **Применить** для просмотра фильтрованных результатов

## Экспорт изображений

### 1. Экспорт одного изображения

1. Выберите изображение
2. Щелкните правой кнопкой мыши и выберите **Экспортировать изображение**
3. Выберите целевую папку
4. Нажмите **Сохранить**

### 2. Экспорт нескольких изображений

1. Выберите несколько изображений
2. Щелкните правой кнопкой мыши и выберите **Экспортировать выбранные изображения**
3. Выберите целевую папку
4. Нажмите **Сохранить**

### 3. Экспорт с метаданными

- При экспорте вы можете выбрать включение метаданных
- Отметьте опцию "Включить метаданные" в диалоговом окне экспорта
- Метаданные будут встроены в экспортированные изображения
