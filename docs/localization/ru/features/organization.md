# Организация

> **Русская версия является официальной**

AVA AIGC Toolbox предоставляет несколько мощных способов организации ваших изображений, созданных с помощью ИИ: папки, альбомы и теги. В этом руководстве описано, как использовать эти функции для поддержания вашей библиотеки изображений хорошо организованной и легкой для навигации.

## Папки

Папки в AVA AIGC Toolbox отражают структуру вашей файловой системы, позволяя вам организовывать изображения на основе их физического расположения на вашем компьютере.

### Добавление папок

#### Шаги для добавления корневой папки:
1. В боковой панели перейдите в представление **Папки**
2. Нажмите кнопку `+` в верхней части боковой панели или перейдите в `Файл > Добавить папку`
3. Выберите папку, которую вы хотите добавить в свою библиотеку
4. Настройте параметры папки:
   - **Включать подпапки**: Включить все изображения из поддиректорий
   - **Отслеживать папку**: Автоматически обнаруживать новые изображения, добавленные в эту папку
   - **Исключить папку**: Исключить эту папку из будущих обновлений библиотеки
5. Нажмите **Добавить**, чтобы добавить папку в вашу библиотеку

### Управление папками

#### Обновление папок
1. В боковой панели щелкните правой кнопкой мыши по папке, которую вы хотите обновить
2. Выберите **Обновить папку** из контекстного меню
3. Приложение просканирует папку на наличие новых, измененных или удаленных изображений

#### Удаление папок
1. В боковой панели щелкните правой кнопкой мыши по папке, которую вы хотите удалить
2. Выберите **Удалить папку** из контекстного меню
3. Подтвердите удаление в диалоговом окне
4. **Примечание**: Это удаляет только папку из библиотеки, а не из вашей файловой системы

#### Свойства папки
1. В боковой панели щелкните правой кнопкой мыши по папке
2. Выберите **Свойства** из контекстного меню
3. В диалоговом окне свойств будут отображены:
   - Путь и размер папки
   - Количество содержащихся изображений
   - Дата последнего сканирования
   - Параметры папки (включать подпапки, отслеживать папку и т.д.)
4. Нажмите **Закрыть**, когда закончите

### Навигация по папкам

#### Раскрытие и сворачивание папок
- Щелкните по стрелке рядом с папкой, чтобы раскрыть или свернуть ее
- Двойной щелчок по папке откроет ее в основной области контента

#### Поиск внутри папок
1. Перейдите в папку, в которой вы хотите искать
2. Используйте строку поиска в верхней части окна
3. Поиск будет ограничен текущей папкой и ее подпапками

## Альбомы

Альбомы - это пользовательские коллекции изображений, которые могут содержать изображения из разных папок. В отличие от папок, альбомы не отражают структуру вашей файловой системы - это виртуальные коллекции.

### Создание альбомов

#### Шаги для создания альбома:
1. В боковой панели перейдите в представление **Альбомы**
2. Нажмите кнопку `+` в верхней части боковой панели
3. Введите имя для вашего альбома
4. Нажмите `Enter` или щелкните за пределами текстового поля, чтобы создать альбом

#### Создание альбомов из выбранных изображений:
1. Выберите одно или несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Добавить в альбом > Новый альбом**
3. Введите имя для альбома
4. Нажмите `Enter`, чтобы создать альбом

### Управление альбомами

#### Добавление изображений в альбомы

##### Добавление одного изображения:
1. Выберите изображение в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Добавить в альбом**
3. Выберите существующий альбом из списка

##### Добавление нескольких изображений:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Добавить в альбом**
3. Выберите существующий альбом из списка или выберите **Новый альбом**

##### Перетаскивание:
1. Выберите одно или несколько изображений в основной области контента
2. Перетащите их в альбом в боковой панели

#### Удаление изображений из альбомов
1. Перейдите в альбом, содержащий изображения
2. Выберите изображения, которые вы хотите удалить
3. Щелкните правой кнопкой мыши и выберите **Удалить из альбома**
4. Подтвердите удаление в диалоговом окне

#### Переименование альбомов
1. В боковой панели щелкните правой кнопкой мыши по альбому, который вы хотите переименовать
2. Выберите **Переименовать альбом** из контекстного меню
3. Введите новое имя
4. Нажмите `Enter`, чтобы подтвердить

#### Удаление альбомов
1. В боковой панели щелкните правой кнопкой мыши по альбому, который вы хотите удалить
2. Выберите **Удалить альбом** из контекстного меню
3. Подтвердите удаление в диалоговом окне
4. **Примечание**: Это удаляет только альбом, а не изображения, которые он содержит

#### Свойства альбома
1. В боковой панели щелкните правой кнопкой мыши по альбому
2. Выберите **Свойства** из контекстного меню
3. В диалоговом окне свойств будут отображены:
   - Имя и описание альбома
   - Количество содержащихся изображений
   - Дата создания и дата последнего изменения
   - Порядок альбома в боковой панели
4. Нажмите **Закрыть**, когда закончите

### Организация альбомов

#### Переупорядочивание альбомов
1. В боковой панели перейдите в представление **Альбомы**
2. Щелкните и перетащите альбом, чтобы изменить его порядок в списке
3. Новый порядок будет сохранен автоматически

#### Создание иерархии альбомов
1. Создайте родительский альбом, как описано выше
2. Создайте дочерние альбомы
3. Щелкните и перетащите дочерний альбом на родительский альбом, чтобы сделать его подальбомом
4. Раскройте родительский альбом, чтобы просмотреть подальбомы
5. Двойной щелчок по родительскому альбому откроет все изображения в родительском и его подальбомах

## Теги

Теги - это ключевые слова, которые вы можете назначить изображениям для их категоризации и легкого поиска. Теги являются одним из самых мощных инструментов организации в AVA AIGC Toolbox.

### Добавление тегов

#### Добавление тегов к одному изображению:
1. Выберите изображение в основной области контента
2. В панели деталей найдите раздел **Теги**
3. Нажмите кнопку `+` рядом со списком тегов
4. Введите имя тега
5. Нажмите `Enter` или щелкните за пределами текстового поля, чтобы добавить тег

#### Добавление тегов к нескольким изображениям:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Добавить теги**
3. Введите имена тегов, разделенные запятыми
4. Нажмите **Добавить**, чтобы применить теги ко всем выбранным изображениям

#### Добавление тегов с помощью сочетания клавиш:
1. Выберите одно или несколько изображений
2. Нажмите `Ctrl/Cmd + T`, чтобы открыть диалоговое окно тегов
3. Введите имена тегов, разделенные запятыми
4. Нажмите `Enter`, чтобы применить теги

### Управление тегами

#### Просмотр изображений с определенным тегом
1. В боковой панели перейдите в представление **Теги**
2. Щелкните по тегу, чтобы просмотреть все изображения с этим тегом
3. Основная область контента будет отображать только изображения с выбранным тегом

#### Удаление тегов из изображений
1. Выберите изображение в основной области контента
2. В панели деталей найдите раздел **Теги**
3. Наведите курсор на тег, который вы хотите удалить
4. Нажмите кнопку `×`, которая появится рядом с именем тега

#### Удаление тегов из нескольких изображений
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Удалить теги**
3. Выберите теги, которые вы хотите удалить
4. Нажмите **Удалить**, чтобы удалить теги из всех выбранных изображений

#### Переименование тегов
1. В боковой панели перейдите в представление **Теги**
2. Щелкните правой кнопкой мыши по тегу, который вы хотите переименовать
3. Выберите **Переименовать тег** из контекстного меню
4. Введите новое имя
5. Нажмите `Enter`, чтобы подтвердить
6. **Примечание**: Это переименует тег для всех изображений, на которых он используется

#### Удаление тегов
1. В боковой панели перейдите в представление **Теги**
2. Щелкните правой кнопкой мыши по тегу, который вы хотите удалить
3. Выберите **Удалить тег** из контекстного меню
4. Подтвердите удаление в диалоговом окне
5. **Примечание**: Это удалит тег из всех изображений

#### Объединение тегов
1. В боковой панели перейдите в представление **Теги**
2. Щелкните правой кнопкой мыши по тегу, который вы хотите объединить
3. Выберите **Объединить с** из контекстного меню
4. Выберите тег, с которым вы хотите объединить
5. Нажмите **Объединить**, чтобы объединить теги
6. **Примечание**: Все изображения с исходным тегом теперь будут иметь объединенный тег вместо него

### Организация тегов

#### Представление облака тегов
1. В боковой панели перейдите в представление **Теги**
2. Нажмите кнопку **Облако тегов** в верхней части боковой панели
3. Теги отображаются в визуальном облаке, при этом чаще используемые теги появляются большими
4. Щелкните по тегу в облаке, чтобы просмотреть все изображения с этим тегом

#### Сортировка тегов
1. В боковой панели перейдите в представление **Теги**
2. Нажмите кнопку **Сортировать по** в верхней части боковой панели
3. Выберите из:
   - **Имя**: Сортировать теги по алфавиту
   - **Использование**: Сортировать теги по количеству изображений, на которых они используются
   - **Дата создания**: Сортировать теги по дате их создания

#### Группы тегов
1. В боковой панели перейдите в представление **Теги**
2. Нажмите кнопку **Группы тегов** в верхней части боковой панели
3. Создайте группы тегов, нажав **+ Группа**
4. Перетащите теги в группы для их организации
5. Группы тегов помогают вам управлять большим количеством тегов

## Умные коллекции

Умные коллекции - это автоматически генерируемые коллекции изображений на основе определенных критериев. Они обновляются динамически по мере изменения вашей библиотеки.

### Умные коллекции по умолчанию

Приложение включает в себя несколько умных коллекций по умолчанию:

- **Все изображения**: Отображает все изображения в вашей библиотеке
- **Избранное**: Отображает изображения, помеченные как избранные
- **Недавно добавленные**: Отображает изображения, добавленные за последние 30 дней
- **Недавно просмотренные**: Отображает изображения, просмотренные за последние 7 дней
- **Нет тегов**: Отображает изображения без тегов
- **Для удаления**: Отображает изображения, помеченные для удаления

### Создание пользовательских умных коллекций

#### Шаги для создания пользовательской умной коллекции:
1. В боковой панели перейдите в представление **Умные коллекции**
2. Нажмите кнопку `+` в верхней части боковой панели
3. Введите имя для вашей умной коллекции
4. Настройте критерии фильтрации:
   - **Текстовый поиск**: Изображения, совпадающие с определенным текстом в имени файла, тегах или метаданных
   - **Датапазон**: Изображения, созданные или измененные в определенный диапазон дат
   - **Размеры**: Изображения с определенной шириной и высотой
   - **Рейтинг**: Изображения с определенным звездным рейтингом
   - **Теги**: Изображения с определенными тегами
   - **Метаданные ИИ**: Изображения с определенной моделью, сэмплером, шагами и т.д.
5. Нажмите **Сохранить**, чтобы создать умную коллекцию

### Управление умными коллекциями

#### Редактирование умных коллекций
1. В боковой панели щелкните правой кнопкой мыши по умной коллекции, которую вы хотите редактировать
2. Выберите **Редактировать коллекцию** из контекстного меню
3. Измените критерии фильтрации
4. Нажмите **Сохранить**, чтобы обновить умную коллекцию

#### Удаление умных коллекций
1. В боковой панели щелкните правой кнопкой мыши по умной коллекции, которую вы хотите удалить
2. Выберите **Удалить коллекцию** из контекстного меню
3. Подтвердите удаление в диалоговом окне

#### Обновление умных коллекций
1. В боковой панели щелкните правой кнопкой мыши по умной коллекции
2. Выберите **Обновить коллекцию** из контекстного меню
3. Умная коллекция будет обновлена с последними изображениями, соответствующими критериям

## Лучшие практики организации

1. **Используйте последовательную систему именования**: Используйте последовательное именование для папок, альбомов и тегов
2. **Начните с простого**: Начните с базовой структуры организации и расширяйте ее по мере необходимости
3. **Комбинируйте методы**: Используйте папки для физической организации, альбомы для тематической организации и теги для описательной категоризации
4. **Используйте описательные теги**: Используйте конкретные, описательные теги, а не общие
5. **Не переборщайте с тегами**: Используйте 5-10 релевантных тегов на изображение, а не десятки
6. **Регулярно очищайте**: Удаляйте неиспользуемые теги и организуйте вашу библиотеку периодически
7. **Используйте умные коллекции**: Создавайте умные коллекции для часто используемых критериев поиска
8. **Создавайте резервные копии вашей библиотеки**: Регулярно создавайте резервные копии базы данных библиотеки, чтобы сохранить вашу структуру организации

## Устранение проблем с организацией

### Теги не применяются
- **Проверьте выделение**: Убедитесь, что вы выбрали правильные изображения
- **Формат имени тега**: Избегайте использования запятых в именах тегов (запятые используются для разделения нескольких тегов)
- **Проблемы с разрешениями**: Убедитесь, что у вас есть права на запись в базу данных

### Альбомы не показывают изображения
- **Проверьте фильтр альбома**: Убедитесь, что вы не применили фильтр, скрывающий изображения
- **Обновите альбом**: Попробуйте обновить альбом, чтобы обновить его содержимое
- **Проверьте расположение изображения**: Убедитесь, что файлы изображений все еще доступны

### Папки не обновляются
- **Проверьте настройки отслеживания**: Убедитесь, что папка настроена на отслеживание изменений
- **Обновите папку**: Вручную обновите папку, чтобы просканировать ее на наличие изменений
- **Проверьте файловую систему**: Убедитесь, что папка все еще существует и у вас есть доступ к ней
