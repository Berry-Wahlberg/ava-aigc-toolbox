# Поиск и фильтрация

> **Русская версия является официальной**

AVA AIGC Toolbox предоставляет мощные возможности поиска и фильтрации, помогающие вам быстро находить изображения в вашей библиотеке. В этом руководстве описано, как эффективно использовать эти функции.

## Базовый поиск

### Строка поиска

Строка поиска находится в верхней части главного окна и предоставляет быстрый способ поиска изображений:

#### Шаги для выполнения базового поиска:
1. Щелкните в строке поиска или нажмите `Ctrl/Cmd + F`, чтобы переместить фокус на нее
2. Введите ваш поисковый запрос
3. Результаты будут появляться автоматически по мере ввода
4. Нажмите `Enter` для выполнения поиска
5. Нажмите `Esc`, чтобы очистить поиск

### Область поиска

По умолчанию поиск включает в себя:
- Имена файлов
- Теги
- Метаданные ИИ (запрос, отрицательный запрос, модель и т.д.)
- Свойства изображения (рейтинг, статус избранного и т.д.)

### Советы по поиску

- **Используйте кавычки для точных фраз**: "futuristic city" найдет только изображения, содержащие эту точную фразу
- **Используйте операторы AND/OR**: "cat AND dog" найдет изображения, содержащие оба слова, "cat OR dog" найдет изображения, содержащие любое из этих слов
- **Используйте оператор NOT**: "cat NOT dog" найдет изображения, содержащие "cat", но не "dog"
- **Используйте подстановочные знаки**: "cat*" найдет изображения, содержащие слова, начинающиеся с "cat"

## Расширенный поиск

Для более точного контроля над вашим поиском используйте диалоговое окно расширенного поиска:

#### Шаги для открытия расширенного поиска:
1. Нажмите кнопку **Расширенный поиск** рядом со строкой поиска или перейдите в `Правка > Расширенный поиск`
2. Появится диалоговое окно расширенного поиска

### Параметры расширенного поиска

Диалоговое окно расширенного поиска позволяет создавать сложные поисковые запросы с использованием нескольких критериев:

#### Текстовый поиск
- **Имя файла**: Поиск по имени файла или части имени файла
- **Теги**: Поиск по тегам
- **Запрос**: Поиск внутри запроса изображения
- **Отрицательный запрос**: Поиск внутри отрицательного запроса
- **Модель**: Поиск по имени модели или хэшу
- **Сэмплер**: Поиск по имени сэмплера

#### Датапазон
- **Дата создания**: Поиск изображений, созданных в определенный диапазон дат
- **Дата изменения**: Поиск изображений, измененных в определенный диапазон дат
- **Дата добавления**: Поиск изображений, добавленных в библиотеку в определенный диапазон дат

#### Свойства изображения
- **Рейтинг**: Поиск по звездному рейтингу (1-5 звезд)
- **Избранное**: Показать только избранные или не избранные изображения
- **Для удаления**: Показать изображения, помеченные для удаления
- **NSFW**: Показать NSFW или не NSFW изображения
- **Недоступно**: Показать недоступные изображения

#### Размеры изображения
- **Ширина**: Поиск изображений с определенной шириной (в пикселях)
- **Высота**: Поиск изображений с определенной высотой (в пикселях)
- **Соотношение сторон**: Поиск изображений с определенным соотношением сторон
- **Ориентация**: Поиск по ориентации (портретная, ландшафтная, квадратная)

#### Метаданные ИИ
- **Шаги**: Поиск изображений с определенным количеством шагов генерации
- **CFG Scale**: Поиск изображений с определенным значением CFG Scale
- **Seed**: Поиск изображений с определенным значением seed
- **Clip Skip**: Поиск изображений с определенным значением clip skip
- **Hypernetwork**: Поиск изображений, использующих определенную гиперсеть

### Сохранение расширенных поисков

Вы можете сохранить ваши критерии расширенного поиска как умную коллекцию:

1. Настройте ваши критерии поиска в диалоговом окне расширенного поиска
2. Нажмите **Сохранить как умную коллекцию**
3. Введите имя для умной коллекции
4. Нажмите **Сохранить**
5. Умная коллекция появится в боковой панели под **Умные коллекции**

## Панель фильтров

Панель фильтров предоставляет интерактивный способ фильтрации изображений на основе различных критериев:

#### Шаги для открытия панели фильтров:
1. Нажмите кнопку **Фильтр** на панели инструментов или перейдите в `Вид > Переключить панель фильтров`
2. Панель фильтров появится слева от основной области контента

### Категории фильтров

Панель фильтров организована в несколько категорий:

#### Быстрые фильтры
- **Все изображения**: Показать все изображения в текущем представлении
- **Избранное**: Показать только избранные изображения
- **Недавно добавленные**: Показать изображения, добавленные за последние 30 дней
- **Нет тегов**: Показать изображения без тегов
- **Для удаления**: Показать изображения, помеченные для удаления

#### Теги
- Показывает все теги в вашей библиотеке
- Щелкните по тегу, чтобы отфильтровать по этому тегу
- Щелкните по нескольким тегам, чтобы отфильтровать изображения, имеющие все выбранные теги
- Используйте строку поиска, чтобы найти конкретные теги

#### Модели ИИ
- Показывает все модели ИИ, используемые в ваших изображениях
- Щелкните по модели, чтобы отфильтровать по этой модели
- Показывает количество изображений для каждой модели

#### Сэмплеры
- Показывает все сэмплеры, используемые в ваших изображениях
- Щелкните по сэмплеру, чтобы отфильтровать по этому сэмплеру
- Показывает количество изображений для каждого сэмплера

#### Размеры
- Фильтр по ориентации изображения (портретная, ландшафтная, квадратная)
- Фильтр по минимальной/максимальной ширине и высоте

#### Рейтинг
- Фильтр по звездному рейтингу с использованием ползунка или флажков
- Выберите "Без рейтинга", чтобы показать изображения без рейтинга

### Комбинирование фильтров

Вы можете комбинировать несколько фильтров для создания точных поисков:

1. Примените фильтр из одной категории (например, выберите тег)
2. Примените фильтр из другой категории (например, выберите модель)
3. Результаты будут показывать изображения, которые соответствуют всем примененным фильтрам
4. Щелкните по выбранному фильтру снова, чтобы удалить его
5. Нажмите **Очистить все**, чтобы удалить все фильтры

## Параметры сортировки

Вы можете сортировать ваши изображения, чтобы их было легче просматривать:

#### Шаги для изменения сортировки:
1. Нажмите кнопку **Сортировать** на панели инструментов или перейдите в `Вид > Сортировать по`
2. Выберите параметр сортировки из списка
3. Щелкните снова, чтобы переключиться между порядком по возрастанию и по убыванию

### Доступные параметры сортировки

- **Имя**: Сортировать по имени файла
- **Дата создания**: Сортировать по дате создания изображения
- **Дата изменения**: Сортировать по дате изменения изображения
- **Дата добавления**: Сортировать по дате добавления в библиотеку
- **Размер**: Сортировать по размеру файла
- **Размеры**: Сортировать по размерам изображения
- **Рейтинг**: Сортировать по звездному рейтингу
- **Случайно**: Сортировать изображения случайным образом
- **Шаги**: Сортировать по количеству шагов генерации
- **CFG Scale**: Сортировать по значению CFG Scale
- **Seed**: Сортировать по значению seed

## Умные коллекции

Умные коллекции - это автоматически обновляемые коллекции изображений на основе определенных критериев. Это отличный способ сохранить часто используемые поиски и фильтры.

Дополнительную информацию об умных коллекциях см. в разделе [Организация](./organization.md#smart-collections).

## История поиска

Приложение сохраняет историю ваших недавних поисков:

#### Шаги для доступа к истории поиска:
1. Нажмите стрелку вниз рядом со строкой поиска
2. Появится выпадающее меню, показывающее недавние поиски
3. Щелкните по предыдущему поиску, чтобы выполнить его снова
4. Нажмите **Очистить историю**, чтобы удалить все предыдущие поиски

## Поиск в конкретных представлениях

Вы можете выполнять поиски в конкретных представлениях:

### Поиск в папках
1. Перейдите в папку в боковой панели
2. Выполните поиск
3. Поиск будет ограничен этой папкой и ее подпапками

### Поиск в альбомах
1. Перейдите в альбом в боковой панели
2. Выполните поиск
3. Поиск будет ограничен этим альбомом

### Поиск по тегам
1. Щелкните по тегу в боковой панели
2. Выполните поиск
3. Поиск будет ограничен изображениями с этим тегом

## Советы по эффективному поиску

1. **Начните с базового поиска**: Начните с простого поискового запроса, прежде чем перейти к расширенному поиску
2. **Используйте конкретные термины**: Будьте конкретны с вашими поисковыми терминами для более точных результатов
3. **Комбинируйте фильтры**: Используйте несколько фильтров для сужения результатов
4. **Сохраняйте полезные поиски**: Сохраняйте часто используемые поиски как умные коллекции
5. **Используйте подстановочные знаки**: Используйте подстановочные знаки для более широких поисков
6. **Проверьте область поиска**: Имейте в виду текущую область поиска (все изображения, папка, альбом и т.д.)
7. **Очищайте фильтры**: Помните, что нужно очищать фильтры при начале нового поиска
8. **Используйте историю поиска**: Переиспользуйте предыдущие поиски из истории поиска

## Устранение проблем с поиском

### Никаких результатов не найдено
- **Проверьте поисковый запрос**: Убедитесь, что ваш поисковый запрос правильный и не слишком специфичный
- **Проверьте область поиска**: Убедитесь, что вы ищете в правильном представлении (все изображения, папка, альбом и т.д.)
- **Проверьте фильтры**: Убедитесь, что никакие фильтры не ограничивают ваши результаты
- **Проверьте орфографию**: Убедитесь, что ваши поисковые термины написаны правильно

### Слишком много результатов
- **Сужьте ваш поиск**: Добавьте больше поисковых терминов или примените дополнительные фильтры
- **Используйте более конкретные термины**: Используйте более конкретные термины для сужения результатов
- **Используйте оператор AND**: Используйте "AND", чтобы требовать нескольких терминов

### Неожиданные результаты
- **Проверьте параметры поиска**: Убедитесь, что вы ищете в правильных полях
- **Проверьте использование операторов**: Убедитесь, что вы используете операторы поиска правильно
- **Проверьте использование кавычек**: Убедитесь, что вы используете кавычки для точных фраз правильно

### Медленная производительность поиска
- **Оптимизируйте базу данных**: Перейдите в `Настройки > Библиотека > Оптимизировать базу данных`, чтобы улучшить производительность поиска
- **Сократите область поиска**: Ищите в конкретной папке или альбоме вместо всех изображений
- **Используйте меньше критериев**: Используйте меньше критериев поиска для более быстрых результатов
