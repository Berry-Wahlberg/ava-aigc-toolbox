# Экспорт и совместное использование

> **Русская версия является официальной**

AVA AIGC Toolbox предоставляет несколько вариантов для экспорта и совместного использования ваших изображений, созданных с помощью ИИ. В этом руководстве описано, как эффективно использовать эти функции.

## Экспорт изображений

### 1. Экспорт одного изображения

#### Шаги для экспорта одного изображения:
1. Выберите изображение, которое вы хотите экспортировать
2. Щелкните правой кнопкой мыши и выберите **Экспортировать изображение** или перейдите в `Файл > Экспорт > Экспортировать выбранные изображения`
3. Появится диалоговое окно экспорта
4. Выберите целевую папку
5. Настройте параметры экспорта
6. Нажмите **Экспортировать**, чтобы экспортировать изображение

### 2. Экспорт нескольких изображений

#### Шаги для экспорта нескольких изображений:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Экспортировать выбранные изображения** или перейдите в `Файл > Экспорт > Экспортировать выбранные изображения`
3. Появится диалоговое окно экспорта
4. Выберите целевую папку
5. Настройте параметры экспорта
6. Нажмите **Экспортировать**, чтобы экспортировать все выбранные изображения
7. Отслеживайте прогресс в диалоговом окне экспорта
8. Нажмите **Готово**, когда закончите

### 3. Экспорт всех изображений

#### Шаги для экспорта всех изображений:
1. Перейдите в `Файл > Экспорт > Экспортировать все изображения`
2. Появится диалоговое окно экспорта
3. Выберите целевую папку
4. Настройте параметры экспорта
5. Нажмите **Экспортировать**, чтобы экспортировать все изображения
6. Отслеживайте прогресс в диалоговом окне экспорта
7. Нажмите **Готово**, когда закончите

## Параметры экспорта

Диалоговое окно экспорта предоставляет несколько вариантов для настройки вашего экспорта:

### 1. Назначение
- **Папка**: Выберите целевую папку для экспортированных изображений
- **Структура подпапок**: Выберите, как обрабатывать подпапки (плоская структура, сохранить исходную структуру, создать новую структуру)

### 2. Именование файлов
- **Оригинальное имя файла**: Использовать оригинальные имена файлов
- **Пользовательский шаблон**: Создать пользовательский шаблон имени файла с использованием переменных
  - `{filename}`: Оригинальное имя файла
  - `{id}`: ID изображения
  - `{model}`: Название модели
  - `{date}`: Дата создания
  - `{time}`: Время создания
  - `{seed}`: Значение seed

### 3. Параметры формата
- **Сохранить оригинальный формат**: Сохранить оригинальный формат файла
- **Конвертировать в формат**: Конвертировать изображения в другой формат (JPEG, PNG, WebP, TIFF, BMP)
- **Качество**: Установить качество для поточных форматов (JPEG, WebP)
- **Уровень сжатия**: Установить уровень сжатия для безпоточных форматов (PNG, TIFF)

### 4. Параметры изменения размера
- **Сохранить оригинальный размер**: Сохранить оригинальные размеры изображения
- **Изменить размер до конкретных размеров**: Изменить размер изображения до конкретной ширины и высоты
- **Сохранять соотношение сторон**: Сохранять оригинальное соотношение сторон при изменении размера
- **Изменить размер по проценту**: Изменить размер изображения на процент от оригинального размера

### 5. Параметры метаданных
- **Включить метаданные**: Встроить метаданные в экспортированные изображения
- **Выбрать поля метаданных**: Выберите, какие поля метаданных включить
- **Формат**: Выберите формат метаданных (EXIF, XMP, Текстовые блоки PNG)
- **Сохранить оригинальные метаданные**: Сохранить все оригинальные метаданные

### 6. Дополнительные параметры
- **Перезаписывать существующие файлы**: Заменять файлы с тем же именем в целевой папке
- **Создавать подпапки для разных моделей**: Создавать отдельные подпапки для разных моделей ИИ
- **Открыть целевую папку после экспорта**: Открыть целевую папку в вашем файловом менеджере после экспорта

## Совместное использование изображений

### 1. Совместное использование по электронной почте

#### Шаги для совместного использования по электронной почте:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **Поделиться > Электронная почта**
3. От откроется ваш почтовый клиент по умолчанию с прикрепленными изображениями
4. Составьте ваше электронное письмо и отправьте

### 2. Совместное использование в социальных сетях

#### Шаги для совместного использования в социальных сетях:
1. Выберите изображение
2. Щелкните правой кнопкой мыши и выберите **Поделиться > Социальные сети**
3. Выберите платформу социальных сетей (если поддерживается)
4. Следуйте инструкциям по совместному использованию для конкретной платформы

### 3. Копирование в буфер обмена

#### Шаги для копирования в буфер обмена:
1. Выберите изображение
2. Щелкните правой кнопкой мыши и выберите **Копировать** или нажмите `Ctrl/Cmd + C`
3. Изображение будет скопировано в ваш буфер обмена
4. Вставьте его в другое приложение с помощью `Ctrl/Cmd + V`

### 4. Генерация ссылки для совместного использования

#### Шаги для генерации ссылки для совместного использования:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **Поделиться > Генерировать ссылку**
3. Настройте параметры совместного использования (срок действия, права доступа)
4. Нажмите **Генерировать ссылку**
5. Скопируйте сгенерированную ссылку для совместного использования с другими

## Пакетный экспорт

Для более расширенных вариантов пакетного экспорта используйте инструмент пакетного экспорта:

#### Шаги для использования пакетного экспорта:
1. Перейдите в `Инструменты > Пакетные операции > Пакетный экспорт`
2. Выберите изображения, которые вы хотите экспортировать (все, текущее представление или выбранные)
3. Настройте параметры экспорта, как описано выше
4. Нажмите **Экспортировать**, чтобы начать пакетный экспорт
5. Отслеживайте прогресс в диалоговом окне пакетного экспорта
6. Нажмите **Готово**, когда закончите

## Предустановки экспорта

Вы можете сохранить конфигурации экспорта как предустановки для быстрого доступа:

### Создание предустановок экспорта

#### Шаги для создания предустановки экспорта:
1. Откройте диалоговое окно экспорта
2. Настройте параметры экспорта
3. Нажмите кнопку **Сохранить предустановку**
4. Введите имя для предустановки
5. Нажмите **Сохранить**, чтобы создать предустановку

### Использование предустановок экспорта

#### Шаги для использования предустановки экспорта:
1. Откройте диалоговое окно экспорта
2. Нажмите раскрывающееся меню **Предустановки**
3. Выберите предустановку из списка
4. Параметры экспорта будут заполнены настройками предустановки
5. Нажмите **Экспортировать**, чтобы использовать предустановку

### Управление предустановками экспорта

#### Шаги для управления предустановками экспорта:
1. Откройте диалоговое окно экспорта
2. Нажмите кнопку **Управление предустановками**
3. В диалоговом окне управления предустановками вы можете:
   - **Редактировать**: Изменять существующие предустановки
   - **Удалять**: Удалять нежелательные предустановки
   - **Изменять порядок**: Изменять порядок предустановок
4. Нажмите **Закрыть**, когда закончите

## Экспорт метаданных

В дополнение к экспорту изображений вы также можете экспортировать метаданные отдельно:

### Экспорт метаданных в файлы

#### Шаги для экспорта метаданных:
1. Выберите одно или несколько изображений
2. Перейдите в `Файл > Экспорт > Экспортировать метаданные`
3. Выберите формат экспорта (JSON, CSV, TXT)
4. Выберите, какие поля метаданных включить
5. Выберите целевую папку
6. Нажмите **Экспортировать**, чтобы экспортировать метаданные

### Форматы метаданных

- **JSON**: Структурированный формат, подходящий для программирования и анализа данных
- **CSV**: Значения, разделенные запятыми, подходящие для электронных таблиц
- **TXT**: Простой текстовый формат, подходящий для чтения

## Лучшие практики для экспорта

1. **Выберите правильный формат**: Используйте соответствующие форматы для вашего использования (JPEG для веб, PNG для безпоточного сжатия, WebP для современного веб)
2. **Оптимизируйте для назначения**: Изменяйте размер и сжимайте изображения соответствующим образом для их предполагаемого использования
3. **Включайте релевантные метаданные**: Всегда включайте релевантные метаданные при совместном использовании с другими
4. **Используйте описательные имена файлов**: Используйте описательные имена файлов для легкой идентификации
5. **Создавайте предустановки экспорта**: Сохраняйте часто используемые конфигурации экспорта как предустановки
6. **Тестируйте с образцами изображений**: Сначала протестируйте ваши настройки экспорта на нескольких образцах изображений
7. **Создавайте резервные копии перед экспортом**: Создавайте резервные копии ваших изображений перед выполнением больших операций экспорта
8. **Отслеживайте прогресс**: Следите за прогрессом экспорта для больших пакетов

## Устранение проблем с экспортом

### Экспорт не работает
- **Проверьте разрешения назначения**: Убедитесь, что у вас есть права на запись в целевую папку
- **Проверьте размер файла**: Некоторые форматы экспорта могут иметь ограничения по размеру
- **Проверьте формат изображения**: Убедитесь, что формат изображения поддерживается для выбранного формата экспорта
- **Проверьте свободное место на диске**: Убедитесь, что на диске достаточно места в целевой папке

### Изображения экспортируются без метаданных
- **Проверьте параметры метаданных**: Убедитесь, что вы выбрали включение метаданных в экспорте
- **Проверьте поддержку формата**: Убедитесь, что формат экспорта поддерживает встраивание метаданных
- **Проверьте доступность метаданных**: Убедитесь, что изображения имеют метаданные для экспорта

### Медленная производительность экспорта
- **Уменьшите качество/сжатие**: Уменьшите качество или сжатие для более быстрого экспорта
- **Экспортируйте меньше изображений**: Экспортируйте изображения в меньших пакетах
- **Закройте другие приложения**: Освободите системные ресурсы, закрыв другие приложения
- **Используйте более быстрые форматы**: Используйте более быстрые форматы, такие как WebP, для веб-экспорта

### Неправильные имена файлов
- **Проверьте шаблон именования**: Убедитесь, что ваш пользовательский шаблон именования правильный
- **Проверьте переменные**: Убедитесь, что вы используете действительные переменные в вашем шаблоне именования
- **Избегайте специальных символов**: Избегайте специальных символов в именах файлов, которые могут не поддерживаться вашей файловой системой
