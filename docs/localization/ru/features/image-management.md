# Управление изображениями

> **Русская версия является официальной**

В этом руководстве описаны основные функции управления изображениями в AVA AIGC Toolbox, включая импорт, просмотр и управление изображениями, созданными с помощью ИИ.

## Импорт изображений

### 1. Импорт из папок

Вы можете импортировать изображения из своей файловой системы в AVA AIGC Toolbox:

#### Шаги для импорта:
1. Нажмите кнопку **Импорт** на панели инструментов или перейдите в `Файл > Импорт > Импортировать изображения`
2. В диалоговом окне импорта нажмите **Добавить папку** и выберите одну или несколько папок, содержащих изображения
3. Настройте параметры импорта:
   - **Включать подпапки**: Импортировать изображения из всех поддиректорий
   - **Перезаписывать существующие изображения**: Заменять изображения с тем же путем в библиотеке
   - **Извлекать метаданные**: Автоматически извлекать метаданные ИИ из изображений
   - **Генерировать эскизы**: Создавать или перегенерировать эскизы для всех импортированных изображений
4. Нажмите **Начать импорт**, чтобы начать процесс импорта
5. Отслеживайте прогресс в диалоговом окне импорта
6. Нажмите **Готово**, когда импорт будет завершен

### 2. Перетаскивание

Вы также можете импортировать изображения, перетаскивая их в приложение:

#### Шаги для импорта через перетаскивание:
1. Откройте файловый менеджер вашей системы (Проводник/Finder)
2. Выберите одно или несколько изображений или папок
3. Перетащите их в окно AVA AIGC Toolbox
4. Появится диалоговое окно подтверждения с параметрами импорта
5. Настройте параметры по желанию
6. Нажмите **Импортировать**, чтобы начать процесс импорта

### 3. Автоматический импорт

Приложение может автоматически мониторить папки на наличие новых изображений:

#### Шаги для настройки автоматического импорта:
1. Перейдите в `Настройки > Библиотека > Отслеживаемые папки`
2. Нажмите **Добавить папку** и выберите папку для отслеживания
3. Настройте параметры отслеживания:
   - **Включать подпапки**: Отслеживать все поддиректории
   - **Интервал проверки**: Как часто проверять наличие новых изображений (в минутах)
   - **Автоматически импортировать новые изображения**: Автоматически импортировать новые изображения при обнаружении
4. Нажмите **Сохранить**, чтобы включить автоматический импорт

## Просмотр изображений

### 1. Сеточный вид

Сеточный вид отображает изображения в сетке эскизов, который является видом по умолчанию при открытии приложения:

#### Навигация в сеточном виде:
- **Прокрутка**: Используйте колесо мыши или полосу прокрутки для навигации по сетке
- **Масштаб**: 
  - Используйте `Ctrl/Cmd +`, чтобы увеличить масштаб
  - Используйте `Ctrl/Cmd -`, чтобы уменьшить масштаб
  - Используйте `Ctrl/Cmd 0`, чтобы сбросить масштаб
- **Настройка размера эскизов**: Используйте ползунок на панели инструментов или перейдите в `Вид > Размер эскизов`

#### Выбор изображений в сеточном виде:
- **Одно изображение**: Щелкните на изображении, чтобы его выбрать
- **Несколько изображений**: 
  - Удерживайте `Ctrl/Cmd` и щелкайте, чтобы выбрать несколько отдельных изображений
  - Удерживайте `Shift` и щелкайте, чтобы выбрать диапазон изображений
  - Щелкните и перетащите, чтобы нарисовать прямоугольник выбора вокруг нескольких изображений
- **Выбрать все**: Нажмите `Ctrl/Cmd + A`, чтобы выбрать все изображения в текущем представлении
- **Снять выделение**: Нажмите `Esc` или перейдите в `Правка > Снять выделение`

### 2. Списочный вид

Списочный вид отображает изображения в списке с подробными информационными колонками:

#### Переключение на списочный вид:
- Нажмите кнопку **Списочный вид** на панели инструментов или перейдите в `Вид > Списочный вид`

#### Настройка списочного вида:
- **Сортировка по колонкам**: Щелкните по заголовку колонки, чтобы отсортировать по этой колонке
- **Изменение ширины колонок**: Перетащите разделители между заголовками колонок, чтобы настроить ширину
- **Показать/скрыть колонки**: Щелкните правой кнопкой мыши по любому заголовку колонки, чтобы показать или скрыть колонки
- **Сбросить колонки**: Перейдите в `Вид > Сбросить колонки списка`, чтобы восстановить колонки по умолчанию

#### Доступные колонки:
- Имя файла
- Путь
- Размер
- Размеры
- Дата добавления
- Дата изменения
- Рейтинг
- Статус избранного
- Теги
- Модель
- Сэмплер
- Шаги
- CFG Scale
- Seed

### 3. Полноэкранный режим

Полноэкранный режим позволяет просматривать изображения в полном разрешении:

#### Открытие полноэкранного режима:
- Двойной щелчок на изображении в сеточном или списочном виде
- Выберите изображение и нажмите `Enter`
- Щелчок правой кнопкой мыши на изображении и выберите **Просмотр**

#### Навигация в полноэкранном режиме:
- **Следующее изображение**: Используйте `Right Arrow`, `Page Down` или щелкните по кнопке со стрелкой вправо
- **Предыдущее изображение**: Используйте `Left Arrow`, `Page Up` или щелкните по кнопке со стрелкой влево
- **Масштаб**: 
  - Используйте колесо мыши для масштабирования вверх/вниз
  - Щелкните и перетащите, чтобы панорамировать при увеличении
  - Нажмите `F11`, чтобы переключаться между режимом подгонки под экран и реальным размером
- **Выход из полноэкранного режима**: Нажмите `Esc` или щелкните по кнопке закрытия

#### Параметры полноэкранного режима:
- **Показать/скрыть элементы управления**: Переместите мышь в нижнюю часть экрана, чтобы показать элементы управления
- **Информация об изображении**: Щелкните по кнопке `i`, чтобы переключить отображение информации об изображении
- **Слайд-шоу**: Щелкните по кнопке воспроизведения, чтобы начать слайд-шоу
- **Добавить в избранное**: Щелкните по кнопке с сердечком, чтобы добавить/удалить из избранного

## Операции с изображениями

### 1. Копирование изображений

Вы можете копировать изображения в буфер обмена или в другие места:

#### Копирование в буфер обмена:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **Копировать**
3. Вставьте изображения в другое приложение или место с помощью `Ctrl/Cmd + V`

#### Копирование в папку:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **Копировать в**
3. Выберите целевую папку
4. Нажмите **Копировать**, чтобы скопировать изображения

### 2. Перемещение изображений

Вы можете перемещать изображения в разные папки внутри своей библиотеки:

#### Шаги для перемещения изображений:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **Переместить в**
3. Выберите целевую папку
4. Нажмите **Переместить**, чтобы переместить изображения

### 3. Переименование изображений

Вы можете переименовать отдельные изображения или несколько изображений одновременно:

#### Переименование одного изображения:
1. Выберите изображение
2. Щелкните правой кнопкой мыши и выберите **Переименовать**
3. Введите новое имя
4. Нажмите `Enter`, чтобы подтвердить

#### Пакетное переименование изображений:
1. Выберите несколько изображений
2. Перейдите в `Инструменты > Пакетные операции > Пакетное переименование`
3. Настройте параметры переименования:
   - **Префикс**: Добавить префикс к именам файлов
   - **Суффикс**: Добавить суффикс к именам файлов
   - **Нумерация**: Добавить последовательные номера
   - **Шаблон**: Использовать пользовательский шаблон для переименования
4. Предварительно просмотрите новые имена
5. Нажмите **Переименовать**, чтобы применить изменения

### 4. Удаление изображений

Вы можете удалить изображения из своей библиотеки:

#### Удаление одного изображения:
1. Выберите изображение
2. Нажмите `Delete` или щелкните правой кнопкой мыши и выберите **Удалить**
3. Подтвердите удаление в диалоговом окне

#### Удаление нескольких изображений:
1. Выберите несколько изображений
2. Нажмите `Delete` или щелкните правой кнопкой мыши и выберите **Удалить**
3. Подтвердите удаление в диалоговом окне

#### Пометка для удаления:
Вы можете пометить изображения для удаления без их немедленного удаления:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **Пометить для удаления**
3. Чтобы просмотреть помеченные изображения, перейдите в **Умные коллекции > Для удаления**
4. Чтобы удалить помеченные изображения, выберите их и нажмите **Удалить**
5. Чтобы снять пометку, щелкните правой кнопкой мыши и выберите **Снять пометку для удаления**

### 5. Свойства изображения

Вы можете просмотреть подробные свойства любого изображения:

#### Шаги для просмотра свойств изображения:
1. Выберите изображение
2. Щелкните правой кнопкой мыши и выберите **Свойства**
3. В диалоговом окне свойств будут отображены:
   - Основная информация (имя файла, путь, размер, размеры)
   - Атрибуты файла (дата создания, дата изменения, дата доступа)
   - Метаданные ИИ (запрос, отрицательный запрос, шаги и т.д.)
   - Статистика изображения (цветовой профиль, битовая глубина)
4. Нажмите **Закрыть**, когда закончите

## Управление эскизами

### 1. Генерация эскизов

Приложение автоматически генерирует эскизы при импорте изображений, но вы можете перегенерировать их при необходимости:

#### Шаги для перегенерации эскизов:
1. Выберите одно или несколько изображений или перейдите в представление папки/альбома
2. Перейдите в `Инструменты > Инструменты ИИ > Сгенерировать эскизы`
3. Настройте параметры:
   - **Перегенерировать все эскизы**: Заменить существующие эскизы
   - **Качество**: Установить качество эскизов (высокое, среднее, низкое)
   - **Размер**: Установить максимальный размер эскизов
4. Нажмите **Генерировать**, чтобы начать процесс
5. Отслеживайте прогресс в диалоговом окне
6. Нажмите **Готово**, когда закончите

### 2. Настройки эскизов

Вы можете настроить параметры эскизов в настройках приложения:

#### Шаги для настройки параметров эскизов:
1. Перейдите в `Настройки > Отображение > Эскизы`
2. Настройте параметры:
   - **Размер эскизов по умолчанию**: Установить размер по умолчанию для сеточного вида
   - **Качество**: Установить качество эскизов (высокое, среднее, низкое)
   - **Размер кэша**: Установить максимальный размер кэша для эскизов
   - **Автоматическая перегенерация**: Автоматически перегенерировать эскизы при необходимости
3. Нажмите **Сохранить**, чтобы применить изменения

## Лучшие практики управления изображениями

1. **Организуйте сначала**: Планируйте структуру папок перед импортом большого количества изображений
2. **Используйте описательные имена**: Называйте папки и изображения так, чтобы их было легко найти
3. **Импортируйте с метаданными**: Всегда извлекайте метаданные при импорте изображений
4. **Регулярные резервные копии**: Регулярно создавайте резервные копии базы данных библиотеки (см. [Резервное копирование и восстановление](../advanced-features/backup-restore.md))
5. **Оптимизируйте производительность**: Для больших библиотек рассмотрите возможность увеличения размера кэша и использования более низкого качества эскизов
6. **Регулярно очищайте**: Удаляйте неиспользуемые изображения и организуйте свою библиотеку периодически
7. **Используйте умные коллекции**: Воспользуйтесь умными коллекциями для быстрого доступа к часто используемым группам изображений

## Устранение проблем с управлением изображениями

### Изображения не импортируются
- **Проверьте разрешения файлов**: Убедитесь, что у вас есть права на чтение исходных папок
- **Поддерживаемые форматы**: Убедитесь, что ваши изображения находятся в поддерживаемых форматах (JPEG, PNG, WebP, TIFF, BMP, GIF)
- **Размер файла**: Проверьте, не слишком ли большие изображения для импорта
- **Поврежденные файлы**: Некоторые изображения могут быть повреждены - попробуйте импортировать одно изображение за раз, чтобы определить проблемные файлы

### Эскизы не отображаются
- **Перегенерируйте эскизы**: Попробуйте перегенерировать эскизы для затронутых изображений
- **Проверьте кэш**: Очистите кэш эскизов в `Настройки > Отображение > Эскизы > Очистить кэш`
- **Доступ к файлам**: Убедитесь, что приложению есть доступ к файлам изображений

### Медленная загрузка изображений
- **Увеличьте размер кэша**: Увеличьте размер кэша эскизов в настройках
- **Уменьшите качество эскизов**: Уменьшите качество эскизов для лучшей производительности
- **Закройте другие приложения**: Освободите системные ресурсы, закрыв другие приложения
- **Оптимизируйте базу данных**: Используйте инструмент оптимизации базы данных в `Настройки > Библиотека > Оптимизировать базу данных`
