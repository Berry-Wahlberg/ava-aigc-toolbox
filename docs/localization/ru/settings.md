# Настройки

> **Русская версия является официальной**

AVA AIGC Toolbox предоставляет полный набор настроек для настройки вашего опыта. Это руководство охватывает все доступные настройки и способы их конфигурации.

## Доступ к настройкам

#### Шаги для открытия настроек:
1. Нажмите кнопку **Настройки** на панели инструментов или перейдите в `Файл > Настройки`
2. Откроется диалоговое окно настроек
3. Используйте боковую панель для навигации между различными категориями настроек
4. Нажмите **Сохранить**, чтобы применить изменения, или **Отмена**, чтобы отбросить изменения
5. Некоторые настройки могут потребовать перезапуска приложения для вступления в силу

## Общие настройки

### Приложение
- **Язык**: Выберите язык приложения (Английский и т.д.)
- **Тема**: Выберите между светлой, темной или системной темой
- **Поведение при запуске**: Настройте, что происходит при запуске приложения
  - **Показывать приветственный экран**: Показать приветственный экран при запуске
  - **Открыть последнюю библиотеку**: Автоматически открывать последнюю использованную библиотеку
  - **Свернуть в трей**: Запускать свернутым в системный трей
- **Проверки обновлений**: Настройте автоматические проверки обновлений
  - **Автоматически проверять обновления**: Включить/отключить автоматические проверки обновлений
  - **Канал обновлений**: Выберите канал обновлений (стабильный, бета, ночной)

### Интерфейс
- **Размер шрифта**: Настройте размер шрифта приложения для лучшей читаемости
- **Размер иконок**: Изменить размер иконок в интерфейсе
- **Анимации**: Включить/отключить анимации
- **Подсказки**: Включить/отключить подсказки
- **Строка статуса**: Показать/скрыть строку статуса

## Сброс настроек

### Сброс к значениям по умолчанию
1. Перейдите в `Настройки > Расширенные > Сброс настроек`
2. Нажмите **Сбросить к значениям по умолчанию**
3. Подтвердите сброс в диалоговом окне
4. Приложение перезапустится с настройками по умолчанию

### Сброс определенных настроек
1. Перейдите в категорию настроек, которую вы хотите сбросить
2. Нажмите кнопку **Сбросить к значениям по умолчанию** внизу страницы
3. Подтвердите сброс в диалоговом окне
4. Нажмите **Сохранить**, чтобы применить изменения

## Импорт/экспорт настроек

### Экспорт настроек
1. Перейдите в `Настройки > Расширенные > Импорт/экспорт настроек`
2. Нажмите **Экспортировать настройки**
3. Выберите место для сохранения файла настроек
4. Нажмите **Сохранить**, чтобы экспортировать настройки

### Импорт настроек
1. Перейдите в `Настройки > Расширенные > Импорт/экспорт настроек`
2. Нажмите **Импортировать настройки**
3. Выберите файл настроек из вашей файловой системы
4. Нажмите **Открыть**, чтобы импортировать настройки
5. Перезапустите приложение, чтобы применить импортированные настройки

## Лучшие практики для настроек

1. **Начните с значений по умолчанию**: Начните с настроек по умолчанию и настройте по мере необходимости
2. **Резервное копирование настроек**: Экспортируйте ваши настройки после внесения значительных изменений
3. **Тестируйте изменения**: Протестируйте изменения настроек перед полноценным использованием
4. **Перезапуск при необходимости**: Всегда перезапускайте приложение, когда предлагается для вступления изменений в силу
5. **Документируйте пользовательские настройки**: Ведите запись пользовательских настроек, которые вы сделали
6. **Используйте предустановки**: Сохраняйте часто используемые конфигурации как предустановки
7. **Оптимизируйте производительность**: Настройте параметры производительности в зависимости от вашей системы
8. **Сбросьте при возникновении проблем**: Если вы сталкиваетесь с проблемами, попробуйте сбросить настройки к значениям по умолчанию

## Устранение проблем с настройками

### Настройки не сохраняются
- **Проверьте разрешения**: Убедитесь, что у вас есть права на запись в месторасположение файла настроек
- **Проверьте свободное место на диске**: Убедитесь, что на диске достаточно места для настроек
- **Закройте другие экземпляры**: Убедитесь, что не запущены другие экземпляры приложения
- **Сбросьте настройки**: Попробуйте сбросить настройки к значениям по умолчанию

### Приложение не запускается после изменения настроек
- **Ручной сброс настроек**: Удалите файл настроек, чтобы сбросить к значениям по умолчанию
  - Windows: `%APPDATA%/AIGenManager/settings.json`
  - macOS: `~/.local/share/AIGenManager/settings.json`
  - Linux: `~/.local/share/AIGenManager/settings.json`
- **Переустановите приложение**: Если ручной сброс не работает, переустановите приложение

### Проблемы с производительностью
- **Настройте параметры кэша**: Увеличьте размеры кэша для лучшей производительности
- **Уменьшите количество параллельных процессов**: Уменьшите количество параллельных процессов
- **Отключите анимации**: Выключите анимации для более быстрой производительности
- **Включите ленивую загрузку**: Включите ленивую загрузку, чтобы сократить время начальной загрузки
