# Обзор пользовательского интерфейса

> **Русская версия является официальной**

AVA AIGC Toolbox имеет интуитивно понятный, хорошо организованный пользовательский интерфейс, предназначенный для эффективного управления вашими изображениями, созданными с помощью ИИ. Это руководство предоставляет подробное объяснение всех основных компонентов интерфейса.

## Основная компоновка окна

Окно приложения разделено на пять основных секций:

1. **Меню бар** - Верхняя навигация и команды приложения
2. **Боковая панель** - Быстрый доступ к различным представлениям и фильтрам
3. **Панель инструментов** - Общие действия и настройки
4. **Основная область контента** - Отображение изображений и их деталей
5. **Строка статуса** - Статус и информация о приложении

## 1. Меню бар

Меню бар содержит команды и настройки для всего приложения, организованные в следующие меню:

### Меню Файл
- **Новая библиотека**: Создать новую пустую библиотеку изображений
- **Открыть библиотеку**: Открыть существующую базу данных библиотеки
- **Импорт**: 
  - **Импортировать изображения**: Импортировать изображения из папок
  - **Импортировать метаданные**: Импортировать метаданные из файлов
- **Экспорт**: 
  - **Экспортировать выбранные изображения**: Экспортировать выбранные изображения в папку
  - **Экспортировать все изображения**: Экспортировать все изображения в текущем представлении
- **Настройки**: Открыть настройки приложения
- **Выход**: Закрыть приложение

### Меню Правка
- **Отменить**: Отменить последнее действие
- **Повторить**: Повторить последнее отмененное действие
- **Выделить все**: Выделить все изображения в текущем представлении
- **Снять выделение**: Снять выделение со всех выбранных изображений
- **Инвертировать выделение**: Инвертировать текущее выделение
- **Найти**: Открыть диалоговое окно поиска

### Меню Вид
- **Переключить боковую панель**: Показать или скрыть боковую панель
- **Переключить панель деталей**: Показать или скрыть панель деталей
- **Режим просмотра**: 
  - **Сеточный вид**: Отображать изображения в сетке
  - **Списочный вид**: Отображать изображения в списке с деталями
- **Сортировать по**: Изменить порядок сортировки изображений
- **Масштаб**: Настроить уровень масштабирования сетки изображений
- **Обновить**: Обновить текущее представление

### Меню Инструменты
- **Пакетные операции**: 
  - **Пакетное переименование**: Переименовать несколько изображений сразу
  - **Пакетное тегирование**: Добавить теги к нескольким изображениям
  - **Пакетный экспорт**: Экспортировать несколько изображений с пользовательскими настройками
- **Редактор метаданных**: Открыть расширенные инструменты редактирования метаданных
- **Инструменты изображений**: 
  - **Обрезать**: Обрезать изображения
  - **Изменить размер**: Изменить размер изображений
  - **Конвертировать формат**: Конвертировать изображения в различные форматы
- **Инструменты ИИ**: 
  - **Автоматическое тегирование**: Использовать ИИ для автоматического тегирования изображений
  - **Сгенерировать эскизы**: Перегенерировать эскизы для всех изображений

### Меню Справка
- **Документация**: Открыть эту документацию
- **Сочетания клавиш**: Показать сочетания клавиш
- **О программе**: Показать версию приложения и авторские права
- **Проверить обновления**: Проверить наличие новых версий
- **Сообщить о проблеме**: Открыть страницу проблем на GitHub

## 2. Боковая панель

Боковая панель предоставляет быстрый доступ к различным представлениям и функциям организации:

### Представление папок
- **Корневые папки**: Отображает корневые папки, которые вы добавили в свою библиотеку
- **Подпапки**: Раскройте папки, чтобы просмотреть их содержимое
- **Добавить папку**: Нажмите кнопку `+`, чтобы добавить новую корневую папку
- **Опции папок**: Щелкните правой кнопкой мыши по папке, чтобы получить доступ к опциям, таким как:
  - Обновить папку
  - Удалить папку
  - Свойства

### Представление альбомов
- **Мои альбомы**: Отображает все созданные пользователем альбомы
- **Добавить альбом**: Нажмите кнопку `+`, чтобы создать новый альбом
- **Опции альбомов**: Щелкните правой кнопкой мыши по альбому, чтобы получить доступ к опциям, таким как:
  - Переименовать альбом
  - Удалить альбом
  - Добавить изображения
  - Свойства

### Представление тегов
- **Все теги**: Отображает все теги в вашей библиотеке, отсортированные по использованию
- **Облако тегов**: Визуальное представление тегов по популярности
- **Добавить тег**: Нажмите кнопку `+`, чтобы создать новый тег
- **Опции тегов**: Щелкните правой кнопкой мыши по тегу, чтобы получить доступ к опциям, таким как:
  - Переименовать тег
  - Удалить тег
  - Объединить теги
  - Просмотреть изображения с тегом

### Умные коллекции
- **Все изображения**: Все изображения в вашей библиотеке
- **Избранное**: Изображения, помеченные как избранные
- **Недавно добавленные**: Изображения, добавленные за последние 30 дней
- **Недавно просмотренные**: Изображения, просмотренные за последние 7 дней
- **Нет тегов**: Изображения без каких-либо тегов
- **Для удаления**: Изображения, помеченные для удаления

## 3. Панель инструментов

Панель инструментов предоставляет быстрый доступ к общим действиям и настройкам:

### Основная панель инструментов
- **Импорт**: Импортировать изображения из папок
- **Обновить**: Обновить текущее представление
- **Режим просмотра**: Переключаться между сеточным и списочным видами
- **Сортировать**: Изменить критерий сортировки (по имени, дате, размеру и т.д.)
- **Фильтр**: Открыть панель фильтров
- **Настройки**: Открыть настройки приложения

### Панель инструментов операций с изображениями
- **Избранное**: Пометить/снять пометку выбранных изображений как избранных
- **Удалить**: Удалить выбранные изображения
- **Тег**: Добавить теги к выбранным изображениям
- **Редактировать**: Открыть редактор изображений
- **Экспортировать**: Экспортировать выбранные изображения

## 4. Основная область контента

Основная область контента отображает изображения и их детали и состоит из двух частей:

### Отображение изображений

#### Сеточный вид
- **Эскизы изображений**: Отображает изображения в сетке эскизов
- **Выделение**: 
  - Щелкните, чтобы выбрать одно изображение
  - Ctrl/Cmd + Щелчок, чтобы выбрать несколько изображений
  - Shift + Щелчок, чтобы выбрать диапазон изображений
  - Перетащите, чтобы выбрать несколько изображений в прямоугольной области
- **Информация об изображении**: Показывает основную информацию при наведении (имя файла, размеры, размер)

#### Списочный вид
- **Колонки**: Отображает изображения с колонками для:
  - Имя файла
  - Размер
  - Размеры
  - Дата добавления
  - Дата изменения
  - Рейтинг
  - Статус избранного
- **Сортировка**: Щелкните по заголовкам колонок, чтобы сортировать по этой колонке
- **Изменяемые колонки**: Перетащите разделители колонок, чтобы настроить ширину

#### Полноэкранный режим
- **Двойной щелчок**: Открыть изображение в полноэкранном режиме
- **Навигация**: 
  - Стрелки для перехода между изображениями
  - Escape для выхода из полноэкранного режима
  - Щелчок правой кнопкой мыши для дополнительных опций
- **Масштаб**: Используйте колесо мыши для масштабирования вверх/вниз
- **Панорамирование**: Щелкните и перетащите, чтобы панорамировать при увеличении

### Панель деталей

Панель деталей появляется справа от окна при выборе изображения и отображает подробную информацию о изображении:

#### Основная информация
- **Имя файла**: Имя файла изображения
- **Путь**: Полный путь к файлу
- **Размер**: Размер файла в байтах/КБ/МБ
- **Размеры**: Ширина и высота в пикселях
- **Разрешение**: Информация о DPI (если доступно)
- **Формат**: Формат файла (JPEG, PNG и т.д.)
- **Дата добавления**: Когда изображение было добавлено в библиотеку
- **Дата изменения**: Дата последнего изменения файла

#### Метаданные ИИ
- **Запрос**: Запрос, использованный для генерации изображения
- **Отрицательный запрос**: Отрицательный запрос, использованный
- **Шаги**: Количество шагов генерации
- **Сэмплер**: Название используемого сэмплера
- **CFG Scale**: Значение CFG Scale
- **Seed**: Значение seed, использованное для генерации
- **Модель**: Название используемой модели
- **Хэш модели**: Хэш модели
- **Ширина/Высота**: Генерируемые размеры

#### Свойства изображения
- **Рейтинг**: Система оценки от 1 до 5 звезд
- **Избранное**: Переключить статус избранного
- **Для удаления**: Пометить для удаления
- **NSFW**: Пометить как небезопасное для работы
- **Недоступно**: Файл недоступен

#### Теги
- **Список тегов**: Отображает все теги, связанные с изображением
- **Добавить тег**: Нажмите `+`, чтобы добавить новые теги
- **Удалить тег**: Нажмите `×`, чтобы удалить существующие теги

## 5. Строка статуса

Строка статуса находится внизу окна и отображает:

- **Всего изображений**: Количество изображений в текущем представлении
- **Выбрано изображений**: Количество выбранных изображений
- **Статус фильтра**: Текущий применяемый фильтр
- **Статус сортировки**: Текущие критерии сортировки
- **Статус приложения**: Текущая активность приложения (импорт, экспорт и т.д.)
- **Размер базы данных**: Размер текущей базы данных

## 6. Диалоги и панели

### Диалог импорта
- **Выбор папки**: Выберите папки, из которых нужно импортировать изображения
- **Опции импорта**: 
  - Включить подпапки
  - Перезаписать существующие изображения
  - Извлечь метаданные
  - Сгенерировать эскизы
- **Индикатор прогресса**: Показывает прогресс импорта

### Диалог экспорта
- **Целевая папка**: Выберите, куда экспортировать изображения
- **Опции экспорта**: 
  - Включить метаданные
  - Изменить размер изображений
  - Конвертировать в формат
  - Переименовать файлы
- **Индикатор прогресса**: Показывает прогресс экспорта

### Панель фильтров
- **Текстовый поиск**: Поиск по имени файла, тегам или метаданным
- **Датапазон**: Фильтровать по дате создания или изменения
- **Размеры**: Фильтровать по ширине и высоте изображения
- **Рейтинг**: Фильтровать по звездному рейтингу
- **Теги**: Фильтровать по конкретным тегам
- **Метаданные ИИ**: Фильтровать по модели, сэмплеру, шагам и т.д.

### Диалог настроек
- **Общие**: Язык приложения, тема и параметры запуска
- **Библиотека**: Расположение базы данных и настройки резервного копирования
- **Импорт**: Параметры импорта по умолчанию
- **Отображение**: Размер эскизов, интервал сетки и параметры просмотра
- **Метаданные**: Параметры извлечения и отображения метаданных
- **Сочетания клавиш**: Настраивать сочетания клавиш

## 7. Контекстные меню

Контекстные меню появляются при щелчке правой кнопкой мыши на различных элементах:

### Контекстное меню изображения
- **Просмотр**: Открыть в полноэкранном режиме
- **Редактировать**: Редактировать изображение или метаданные
- **Копировать**: Скопировать изображение в буфер обмена
- **Переместить в**: Переместить изображение в другую папку или альбом
- **Копировать в**: Скопировать изображение в другое место
- **Удалить**: Удалить изображение из библиотеки
- **Добавить в альбом**: Добавить в существующий альбом
- **Добавить теги**: Добавить теги к изображению
- **Удалить теги**: Удалить теги из изображения
- **Установить рейтинг**: Установить звездный рейтинг
- **Пометить как избранное**: Переключить статус избранного
- **Свойства**: Просмотреть детальные свойства

### Контекстное меню папки
- **Открыть в проводнике/Finder**: Открыть папку в системном файловом менеджере
- **Обновить**: Обновить содержимое папки
- **Удалить папку**: Удалить из библиотеки (не удаляет файлы)
- **Свойства**: Просмотреть свойства папки

### Контекстное меню альбома
- **Открыть**: Просмотреть содержимое альбома
- **Переименовать**: Переименовать альбом
- **Удалить**: Удалить альбом
- **Добавить изображения**: Добавить изображения в альбом
- **Удалить изображения**: Удалить выбранные изображения из альбома
- **Свойства**: Просмотреть свойства альбома

### Контекстное меню тега
- **Просмотреть изображения**: Просмотреть все изображения с этим тегом
- **Переименовать**: Переименовать тег
- **Удалить**: Удалить тег
- **Объединить с**: Объединить с другим тегом
- **Свойства**: Просмотреть свойства тега

## 8. Сочетания клавиш

Для быстрого доступа к общим командам обратитесь к справочнику [Сочетания клавиш](./keyboard-shortcuts.md).

## Параметры настройки

### Тема
- **Светлая тема**: Яркая цветовая схема
- **Темная тема**: Темная цветовая схема
- **Тема системы**: Следовать настройкам темы системы

### Параметры просмотра
- **Размер эскизов**: Настраивать размер эскизов в сеточном виде
- **Интервал сетки**: Настраивать интервал между изображениями в сеточном виде
- **Показать/скрыть колонки**: Настраивать, какие колонки отображаются в списочном виде
- **Положение панели деталей**: Переместить панель деталей влево или вправо

### Размер шрифта
- Настраивать размер шрифта для лучшей читаемости

## Советы по эффективной навигации

1. **Навигация с клавиатуры**: Используйте сочетания клавиш для быстрой работы
2. **Настройте панель инструментов**: Добавьте часто используемые команды на панель инструментов
3. **Закрепите часто используемые элементы**: Закрепите часто используемые папки, альбомы и теги в верхней части соответствующих списков
4. **Используйте умные коллекции**: Воспользуйтесь предустановленными умными коллекциями для быстрого доступа
5. **Пользовательские фильтры**: Создавайте и сохраняйте пользовательские фильтры для повторяющихся поисков
6. **Фокус клавиатуры**: Нажмите `Tab`, чтобы перемещаться между элементами интерфейса
7. **Контекстные меню**: Щелчок правой кнопкой мыши на элементах для быстрого доступа к опциям
