# Интеграция ИИ

> **Русская версия является официальной**

AVA AIGC Toolbox предлагает несколько функций, поддерживаемых искусственным интеллектом, для улучшения вашего рабочего процесса с изображениями, созданными ИИ. Это руководство описывает, как использовать эти расширенные функции интеграции ИИ.

## Маркировка с помощью ИИ

### Автоматическая генерация тегов

Приложение может автоматически генерировать теги для ваших изображений с помощью ИИ:

#### Шаги для автоматической маркировки изображений:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **AI Tools > Auto-Tag Images** или перейдите в `Инструменты > AI Tools > Auto-Tag Images`
3. В диалоговом окне автоматической маркировки выберите модель генерации тегов (если доступно несколько)
4. Выберите параметры тегов:
   - **Number of Tags**: Сколько тегов генерировать на изображение
   - **Tag Confidence Threshold**: Минимальный уровень уверенности для сгенерированных тегов
   - **Overwrite Existing Tags**: Заменить существующие теги сгенерированными тегами
5. Нажмите **Generate Tags**, чтобы начать процесс
6. Отслеживайте прогресс в диалоговом окне
7. Просмотрите сгенерированные теги
8. Нажмите **Apply**, чтобы применить теги к изображениям

### Пользовательские модели тегов

Вы можете использовать пользовательские модели тегов для более специфических потребностей в маркировке:

#### Шаги для добавления пользовательской модели тегов:
1. Перейдите в `Настройки > AI > Модели тегов`
2. Нажмите **Добавить модель**
3. Выберите файл модели из вашей файловой системы
4. Введите имя для модели
5. Нажмите **Сохранить**, чтобы добавить модель

### Настройки модели тегов

#### Шаги для настройки моделей тегов:
1. Перейдите в `Настройки > AI > Модели тегов`
2. Выберите модель, которую хотите настроить
3. Отрегулируйте настройки:
   - **Модель по умолчанию**: Установить как модель тегов по умолчанию
   - **Порог уверенности**: Уровень уверенности по умолчанию для этой модели
   - **Язык тегов**: Язык для сгенерированных тегов
   - **Категории тегов**: Включить/отключить конкретные категории тегов
4. Нажмите **Сохранить**, чтобы применить изменения

## Улучшение метаданных с помощью ИИ

### Умное дополнение метаданных

Приложение может автоматически улучшать метаданные ваших изображений с помощью ИИ:

#### Шаги для улучшения метаданных:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **AI Tools > Enhance Metadata** или перейдите в `Инструменты > AI Tools > Enhance Metadata`
3. В диалоговом окне улучшения выберите, какие поля метаданных нужно улучшить:
   - **Улучшение промпта**: Улучшить текст промпта
   - **Генерация негативных промптов**: Генерировать негативные промпты для изображений без них
   - **Классификация стиля**: Добавить теги стиля и классификацию
4. Нажмите **Улучшить метаданные**, чтобы начать процесс
5. Отслеживайте прогресс в диалоговом окне
6. Просмотрите улучшенные метаданные
7. Нажмите **Apply**, чтобы сохранить изменения

### Улучшение промпта

Вы можете улучшить свои промпты с помощью ИИ для получения лучших результатов при последующей генерации изображений:

#### Шаги для улучшения промптов:
1. Выберите изображение с промптом, который вы хотите улучшить
2. В панели деталей найдите раздел **AI Metadata**
3. Нажмите кнопку **Improve Prompt** рядом с полем промпта
4. Приложение сгенерирует улучшенный промпт
5. Просмотрите улучшенный промпт
6. Нажмите **Принять**, чтобы заменить исходный промпт, или **Отменить**, чтобы сохранить исходный

## Анализ изображений с помощью ИИ

### Анализ стиля изображения

Приложение может анализировать изображения для определения их художественного стиля:

#### Шаги для анализа стиля изображения:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **AI Tools > Analyze Style** или перейдите в `Инструменты > AI Tools > Analyze Style`
3. В диалоговом окне анализа стиля выберите модель анализа стиля
4. Нажмите **Analyze Style**, чтобы начать процесс
5. Отслеживайте прогресс в диалоговом окне
6. Просмотрите результаты анализа стиля
7. Нажмите **Apply**, чтобы добавить теги стиля к изображениям

### Оценка качества изображения

Приложение может оценивать качество изображений, созданных ИИ:

#### Шаги для оценки качества изображения:
1. Выберите одно или несколько изображений
2. Щелкните правой кнопкой мыши и выберите **AI Tools > Assess Quality** или перейдите в `Инструменты > AI Tools > Assess Quality`
3. В диалоговом окне оценки качества выберите модель качества
4. Нажмите **Assess Quality**, чтобы начать процесс
5. Отслеживайте прогресс в диалоговом окне
6. Просмотрите оценки качества
7. Нажмите **Apply**, чтобы добавить оценки качества к изображениям

## Настройки интеграции ИИ

### Общие настройки ИИ

#### Шаги для настройки параметров ИИ:
1. Перейдите в `Настройки > AI > Общие`
2. Настройте общие параметры ИИ:
   - **Включить AI-функции**: Включить или отключить AI-функции
   - **Модель ИИ по умолчанию**: Установить модель ИИ по умолчанию для various tasks
   - **Максимальное количество параллельных запросов**: Количество параллельных запросов ИИ
   - **Кэшировать результаты ИИ**: Кэшировать результаты ИИ для более быстрой обработки
3. Нажмите **Сохранить**, чтобы применить изменения

### Интеграция API

Приложение может интегрироваться с внешними API ИИ:

#### Шаги для настройки интеграции API:
1. Перейдите в `Настройки > AI > API`
2. Выберите API, которое хотите настроить
3. Введите учетные данные API:
   - **API Key**: Ваш ключ API для сервиса
   - **API URL**: URL-адрес конечной точки API
   - **Rate Limit**: Максимальное количество запросов в минуту
4. Нажмите **Тест подключения**, чтобы проверить конфигурацию
5. Нажмите **Сохранить**, чтобы применить изменения

### Поддерживаемые сервисы ИИ

Приложение поддерживает интеграцию с различными сервисами ИИ:

- **Локальные модели**: Запускать модели ИИ локально на вашем компьютере
- **OpenAI**: GPT для генерации и анализа текста
- **Stable Diffusion APIs**: Для генерации и анализа изображений
- **Пользовательские API**: Добавьте свои собственные конечные точки сервисов ИИ

## Интеграция с рабочим процессом ИИ

### Интеграция с инструментами генерации ИИ

Приложение может интегрироваться с инструментами генерации изображений с помощью ИИ:

#### Шаги для настройки интеграции рабочего процесса:
1. Перейдите в `Настройки > AI > Интеграция рабочего процесса`
2. Выберите инструмент генерации ИИ, с которым хотите интегрироваться
3. Настройте параметры инструмента:
   - **Путь к инструменту**: Путь к исполняемому файлу инструмента
   - **Настройки по умолчанию**: Настройки по умолчанию для генерации изображений
   - **Папка вывода**: Папка, где сохраняются генерируемые изображения
4. Нажмите **Тестировать интеграцию**, чтобы проверить конфигурацию
5. Нажмите **Сохранить**, чтобы применить изменения

### Быстрая генерация

Генерируйте новые изображения напрямую из приложения:

#### Шаги для генерации изображений:
1. Перейдите в `Инструменты > AI Tools > Generate Images`
2. В диалоговом окне генерации настройте параметры генерации:
   - **Промпт**: Текстовый промпт для генерации изображения
   - **Негативный промпт**: Текст, указывающий, что следует исключить
   - **Модель**: Модель ИИ для использования при генерации
   - **Шаги**: Количество шагов генерации
   - **Сэмплер**: Метод семплирования
   - **Шкала CFG**: Масштаб классификатора
   - **Seed**: Случайное начальное значение для генерации
   - **Ширина/Высота**: Размеры генерируемого изображения
3. Нажмите **Generate**, чтобы начать процесс генерации
4. Отслеживайте прогресс в диалоговом окне
5. Когда генерация завершена, просмотрите сгенерированные изображения
6. Нажмите **Добавить в библиотеку**, чтобы добавить изображения в вашу библиотеку

## Лучшие практики для интеграции ИИ

1. **Используйте локальные модели при возможности**: Локальные модели работают быстрее и более конфиденциально
2. **Оптимизируйте настройки**: Настройте пороги уверенности и параметры в соответствии с вашими потребностями
3. **Проверяйте сгенерированный контент**: Всегда проверяйте сгенерированные AI-теги и метаданные
4. **Кэшируйте результаты**: Включите кэширование для повторяющихся операций AI
5. **Ограничьте параллельные запросы**: Избегайте перегрузки вашей системы или лимитов API
6. **Регулярно обновляйте модели**: Поддерживайте ваши модели ИИ в актуальном состоянии для получения лучших результатов
7. **Используйте соответствующие модели**: Выберите правильную модель для каждой задачи (маркировка, улучшение и т.д.)
8. **Уважайте лимиты API**: Следуйте лимитам API, чтобы избежать сбоев в обслуживании

## Устранение проблем с ИИ

### AI-функции не работают
- **Проверьте доступность моделей**: Убедитесь, что требуемые модели ИИ установлены
- **Проверьте учетные данные API**: Проверьте ключи API и конфигурацию
- **Проверьте системные требования**: Убедитесь, что ваша система соответствует требованиям к обработке ИИ
- **Проверьте настройки брандмауэра**: Убедитесь, что приложению есть доступ к сети для вызовов API

### Медленная обработка ИИ
- **Уменьшите количество параллельных запросов**: Уменьшите количество параллельных запросов ИИ
- **Используйте меньшие модели**: Перейдите на меньшие и более быстрые модели для лучшей производительности
- **Включите кэширование**: Убедитесь, что кэширование результатов ИИ включено
- **Закройте другие приложения**: Освободите системные ресурсы, закрыв другие приложения

### Плохие результаты ИИ
- **Отрегулируйте порог уверенности**: Измените порог уверенности для получения лучших результатов
- **Попробуйте другие модели**: Экспериментируйте с другими моделями ИИ
- **Улучшите качество входных данных**: Убедитесь, что ваши входные изображения имеют хорошее качество
- **Обновите модели**: Используйте последние версии моделей ИИ

### Проблемы с интеграцией API
- **Проверьте ключ API**: Проверьте, что ваш ключ API правильный и имеет достаточные разрешения
- **Проверьте статус API**: Проверьте, работает ли сервис API
- **Проверьте лимиты скорости**: Убедитесь, что вы не превышаете лимиты API
- **Проверьте подключение к сети**: Убедитесь, что у вас стабильное подключение к Интернету
