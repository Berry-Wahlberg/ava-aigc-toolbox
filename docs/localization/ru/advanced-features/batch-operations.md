# Пакетные операции

> **Русская версия является официальной**

AVA AIGC Toolbox позволяет выполнять массовые операции над несколькими изображениями одновременно, экономя ваше время и усилия. В этом руководстве описаны различные пакетные операции, доступные в приложении.

## Пакетное тегирование

### Добавление тегов нескольким изображениям

#### Шаги для пакетного добавления тегов:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Добавить теги** или перейдите в `Инструменты > Пакетные операции > Пакетное добавление тегов`
3. В диалоговом окне пакетного тегирования введите теги, которые вы хотите добавить, разделяя их запятыми
4. Выберите параметры тегирования:
   - **Заменить существующие теги**: Заменить все существующие теги новыми тегами
   - **Добавить к существующим тегам**: Добавить новые теги к существующим тегам
5. Нажмите **Добавить теги**, чтобы применить теги ко всем выбранным изображениям

### Удаление тегов из нескольких изображений

#### Шаги для пакетного удаления тегов:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Удалить теги** или перейдите в `Инструменты > Пакетные операции > Пакетное удаление тегов`
3. В диалоговом окне пакетного удаления тегов выберите теги, которые вы хотите удалить
4. Нажмите **Удалить теги**, чтобы удалить выбранные теги из всех изображений

### Замена тегов в нескольких изображениях

#### Шаги для пакетной замены тегов:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Заменить теги** или перейдите в `Инструменты > Пакетные операции > Пакетная замена тегов`
3. В диалоговом окне пакетной замены тегов введите старый тег и новый тег
4. Нажмите **Заменить теги**, чтобы заменить старый тег новым тегом во всех изображениях

## Пакетное редактирование метаданных

### Обновление метаданных для нескольких изображений

#### Шаги для пакетного редактирования метаданных:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Редактировать метаданные** или перейдите в `Инструменты > Пакетные операции > Пакетное редактирование метаданных`
3. В диалоговом окне пакетного редактирования метаданных выберите поля метаданных, которые вы хотите редактировать
4. Введите новые значения для каждого выбранного поля
5. Выберите, как применить изменения:
   - **Установить**: Заменить существующие значения на новое значение
   - **Добавить**: Добавить новое значение к существующим значениям
   - **Добавить в начало**: Вставить новое значение перед существующими значениями
6. Нажмите **Применить**, чтобы сохранить изменения для всех выбранных изображений

### Применение шаблонов метаданных

#### Шаги для применения шаблонов метаданных к нескольким изображениям:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Применить шаблон метаданных**
3. Выберите шаблон из списка
4. Нажмите **Применить**, чтобы применить шаблон ко всем выбранным изображениям

## Пакетные операции с изображениями

### Изменение размера нескольких изображений

#### Шаги для пакетного изменения размера изображений:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Изменить размер изображений** или перейдите в `Инструменты > Пакетные операции > Пакетное изменение размера`
3. В диалоговом окне пакетного изменения размера настройте параметры изменения размера:
   - **Изменить размер по проценту**: Изменить размер изображений на процент от оригинального размера
   - **Изменить размер до конкретных размеров**: Изменить размер изображений до конкретной ширины и высоты
   - **Сохранять соотношение сторон**: Сохранять оригинальное соотношение сторон при изменении размера
   - **Качество**: Установить качество для изображений с измененным размером
4. Выберите параметры вывода:
   - **Перезаписывать оригинальные изображения**: Заменить оригинальные изображения версиями с измененным размером
   - **Сохранять как новые изображения**: Сохранять изображения с измененным размером как новые файлы
5. Нажмите **Изменить размер**, чтобы начать процесс
6. Отслеживайте прогресс в диалоговом окне
7. Нажмите **Готово**, когда закончите

### Конвертация нескольких изображений

#### Шаги для пакетной конвертации изображений:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Конвертировать изображения** или перейдите в `Инструменты > Пакетные операции > Пакетная конвертация`
3. В диалоговом окне пакетной конвертации выберите целевой формат (JPEG, PNG, WebP, TIFF, BMP)
4. Настройте параметры формата:
   - **Качество**: Установить качество для поточных форматов (JPEG, WebP)
   - **Уровень сжатия**: Установить уровень сжатия для безпоточных форматов (PNG, TIFF)
5. Выберите параметры вывода:
   - **Перезаписывать оригинальные изображения**: Заменить оригинальные изображения конвертированными версиями
   - **Сохранять как новые изображения**: Сохранять конвертированные изображения как новые файлы
6. Нажмите **Конвертировать**, чтобы начать процесс
7. Отслеживайте прогресс в диалоговом окне
8. Нажмите **Готово**, когда закончите

### Переименование нескольких изображений

#### Шаги для пакетного переименования изображений:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Переименовать изображения** или перейдите в `Инструменты > Пакетные операции > Пакетное переименование`
3. В диалоговом окне пакетного переименования настройте параметры переименования:
   - **Префикс**: Добавить префикс к именам файлов
   - **Суффикс**: Добавить суффикс к именам файлов
   - **Нумерация**: Добавить последовательные номера
   - **Шаблон**: Использовать пользовательский шаблон для переименования
4. Предварительно просмотрите новые имена
5. Нажмите **Переименовать**, чтобы применить изменения

## Пакетный рейтинг и свойства

### Рейтинг нескольких изображений

#### Шаги для пакетного присвоения рейтинга изображениям:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Установить рейтинг**
3. Выберите звездный рейтинг (1-5 звезд)
4. Рейтинг будет применен ко всем выбранным изображениям

### Пометка нескольких изображений как избранных

#### Шаги для пакетной пометки как избранных:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Пометить как избранное** или щелкните по кнопке с сердечком на панели инструментов
3. Статус избранного будет переключен для всех выбранных изображений

### Пометка нескольких изображений для удаления

#### Шаги для пакетной пометки для удаления:
1. Выберите несколько изображений в основной области контента
2. Щелкните правой кнопкой мыши и выберите **Пометить для удаления**
3. Выбранные изображения будут помечены для удаления
4. Чтобы просмотреть помеченные изображения, перейдите в **Умные коллекции > Для удаления**
5. Чтобы удалить помеченные изображения, выберите их и нажмите **Удалить**

## Лучшие практики для пакетных операций

1. **Начните с небольшого пакета**: Сначала протестируйте пакетные операции на небольшом количестве изображений
2. **Используйте отмену при возможности**: Многие пакетные операции поддерживают отмену, используйте ее, если сделали ошибку
3. **Создавайте резервные копии перед пакетными операциями**: Всегда создавайте резервные копии ваших изображений перед выполнением разрушительных пакетных операций
4. **Используйте фильтры для выбора изображений**: Используйте поиск и фильтрацию для выбора именно тех изображений, которые вы хотите обработать
5. **Отслеживайте прогресс**: Следите за прогрессом пакетных операций для больших пакетов
6. **Используйте предустановки**: Сохраняйте часто используемые конфигурации пакетных операций как предустановки
7. **Проверяйте результаты**: Всегда проверяйте результаты пакетных операций перед продолжением
8. **Избегайте перезаписи оригинальных файлов**: При возможности сохраняйте обработанные изображения как новые файлы

## Устранение проблем с пакетными операциями

### Пакетная операция не работает
- **Проверьте разрешения файлов**: Убедитесь, что у вас есть права на запись в изображения
- **Проверьте блокировки файлов**: Убедитесь, что изображения не заблокированы другими приложениями
- **Проверьте форматы изображений**: Убедитесь, что все изображения находятся в поддерживаемых форматах
- **Уменьшите размер пакета**: Попробуйте обрабатывать изображения в меньших пакетах

### Медленная обработка пакета
- **Закройте другие приложения**: Освободите системные ресурсы, закрыв другие приложения
- **Уменьшите количество параллельных процессов**: Уменьшите количество параллельных процессов в настройках
- **Используйте настройки более низкого качества**: Используйте настройки более низкого качества для более быстрой обработки
- **Используйте локальное хранилище**: Убедитесь, что изображения хранятся на локальных дисках, а не на сетевых дисках

### Некорректные результаты от пакетных операций
- **Проверьте конфигурацию**: Убедитесь, что вы правильно настроили пакетную операцию
- **Используйте отмену**: Если доступно, используйте отмену для отката операции
- **Восстановите из резервной копии**: Если операция была разрушительной, восстановите из резервной копии
- **Тестируйте на образцах изображений**: Сначала протестируйте операцию на образцах изображений

### Изображения не выбраны правильно
- **Проверьте выделение**: Убедитесь, что вы выбрали правильные изображения
- **Используйте фильтры**: Используйте поиск и фильтрацию для более точного выбора изображений
- **Используйте умные коллекции**: Используйте умные коллекции для сбора изображений для пакетной обработки
