# Руководство по установке

> **Русская версия является официальной**

## Обзор

Это руководство проведет вас через процесс установки AVA AIGC Toolbox на вашу систему. Приложение поддерживает Windows, macOS и Linux.

## Системные требования

### Минимальные требования
- **Операционная система**: Windows 10+, macOS 10.15+ или Linux (Ubuntu 20.04+, Fedora 32+)
- **.NET Runtime**: .NET 7.0 или новее
- **Дисковое пространство**: 100 МБ свободного места
- **Оперативная память**: Минимум 2 ГБ

### Рекомендуемые требования
- **Оперативная память**: 4 ГБ или больше
- **Процессор**: Многопроцессорный ЦПУ
- **Монитор**: Разрешение 1080p или выше

## Методы установки

### 1. Использование установщика (Windows)

1. **Скачайте установщик**
   - Посетите официальный сайт или страницу релизов на GitHub
   - Скачайте последний установщик `.exe` для Windows

2. **Запустите установщик**
   - Дважды кликните по загруженному файлу `.exe`
   - Следуйте инструкциям на экране
   - Выберите каталог установки (рекомендуется использовать стандартный)
   - Выберите, создавать ли ярлыки на рабочем столе и в меню Пуск

3. **Запустите приложение**
   - Нажмите "Готово", чтобы запустить приложение немедленно
   - Или используйте ярлыки на рабочем столе/меню Пуск позже

### 2. Использование менеджера пакетов (macOS/Linux)

#### macOS (Homebrew)
```bash
brew tap ava-aigc-toolbox/tap
brew install ava-aigc-toolbox
```

#### Linux (Snap)
```bash
sudo snap install ava-aigc-toolbox
```

#### Linux (Debian/Ubuntu)
```bash
sudo dpkg -i ava-aigc-toolbox_*.deb
sudo apt-get install -f
```

### 3. Портативная версия (все платформы)

1. **Скачайте архив**
   - Скачайте последний архив `.zip` (Windows) или `.tar.gz` (macOS/Linux)

2. **Распакуйте архив**
   - Распакуйте содержимое в каталог по вашему выбору
   - Установка не требуется

3. **Запустите приложение**
   - Windows: Дважды кликните по `AIGenManager.exe`
   - macOS/Linux: Запустите `./AIGenManager` в терминале

## Установка .NET Runtime

Если у вас не установлен .NET 7.0 Runtime, вы должны сначала установить его:

### Windows
- Скачайте с https://dotnet.microsoft.com/download/dotnet/7.0
- Запустите установщик и следуйте инструкциям

### macOS
```bash
brew install --cask dotnet-sdk
```

### Linux
```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y dotnet-runtime-7.0

# Fedora
sudo dnf install -y dotnet-runtime-7.0
```

## Проверка установки

1. **Запустите приложение**
2. **Проверьте версию**
   - Перейдите в `Помощь` > `О программе`
   - Убедитесь, что версия соответствует той, которую вы скачали

3. **Протестируйте основные функции**
   - Приложение должно запускаться без ошибок
   - Главное окно должно отображаться правильно
   - Вы должны иметь возможность перемещаться по интерфейсу

## Устранение проблем

### Приложение не запускается
- **Проверьте .NET Runtime**: Убедитесь, что у вас установлен правильный .NET Runtime
- **Проверьте системные требования**: Убедитесь, что ваша система соответствует минимальным требованиям
- **Запустите как администратор**: Попробуйте запустить приложение с правами администратора
- **Проверьте логи**: Посмотрите файлы логов в `%APPDATA%/AIGenManager/` (Windows) или `~/.local/share/AIGenManager/` (macOS/Linux)

### Ошибки установки
- **Установщик Windows**: Убедитесь, что у вас есть права на запись в каталог установки
- **Менеджер пакетов**: Проверьте интернет-соединение и попробуйте снова
- **Портативная версия**: Убедитесь, что вы правильно распаковали все файлы

### Проблемы с производительностью
- **Закройте другие приложения**: Освободите системные ресурсы
- **Увеличьте оперативную память**: Рассмотрите возможность обновления оперативной памяти системы
- **Уменьшите разрешение дисплея**: Отрегулируйте настройки дисплея

## Удаление

### Windows (установщик)
1. Перейдите в `Панель управления` > `Программы` > `Программы и компоненты`
2. Выберите "AVA AIGC Toolbox" из списка
3. Нажмите "Удалить" и следуйте инструкциям

### macOS (Homebrew)
```bash
brew uninstall ava-aigc-toolbox
```

### Linux (Snap)
```bash
sudo snap remove ava-aigc-toolbox
```

### Портативная версия
- Просто удалите распакованный каталог
- При желании удалите папку данных приложения:
  - Windows: `%APPDATA%/AIGenManager/`
  - macOS: `~/.local/share/AIGenManager/`
  - Linux: `~/.local/share/AIGenManager/`
