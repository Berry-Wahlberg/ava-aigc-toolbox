# Сочетания клавиш

> **Русская версия является официальной**

AVA AIGC Toolbox предоставляет полный набор сочетаний клавиш, помогающих вам работать более эффективно. Это руководство перечисляет все доступные сочетания клавиш и способы их настройки.

## Общие сочетания

| Сочетание клавиш | Действие |
|------------------|----------|
| `Ctrl/Cmd + O` | Открыть библиотеку |
| `Ctrl/Cmd + S` | Сохранить текущие изменения |
| `Ctrl/Cmd + Q` | Выход из приложения |
| `Ctrl/Cmd + ,` | Открыть настройки |
| `Ctrl/Cmd + H` | Показать/скрыть боковую панель |
| `Ctrl/Cmd + L` | Переместить фокус на строку поиска |
| `Ctrl/Cmd + F` | Найти в текущем представлении |
| `F1` | Открыть документацию |
| `F12` | Переключить режим отладки |

## Настройка сочетаний клавиш

Вы можете настроить сочетания клавиш в соответствии с вашим рабочим процессом:

#### Шаги для настройки сочетаний клавиш:
1. Перейдите в `Файл > Настройки > Сочетания клавиш`
2. В настройках сочетаний клавиш вы можете:
   - **Просмотреть** все доступные сочетания
   - **Искать** конкретные сочетания
   - **Редактировать** существующие сочетания
   - **Сбросить** сочетания к значениям по умолчанию
3. Чтобы отредактировать сочетание:
   - Нажмите на поле сочетания рядом с действием
   - Нажмите новую комбинацию клавиш
   - Нажмите **Сохранить**, чтобы применить новое сочетание
4. Нажмите **Сохранить**, чтобы сохранить все изменения
5. Перезапустите приложение, чтобы изменения вступили в силу

## Импорт/экспорт конфигураций сочетаний

#### Шаги для экспорта конфигурации сочетаний:
1. Перейдите в `Файл > Настройки > Сочетания клавиш`
2. Нажмите **Экспортировать конфигурацию**
3. Выберите целевую папку для файла экспорта
4. Введите имя файла
5. Нажмите **Сохранить**, чтобы экспортировать конфигурацию

#### Шаги для импорта конфигурации сочетаний:
1. Перейдите в `Файл > Настройки > Сочетания клавиш`
2. Нажмите **Импортировать конфигурацию**
3. Выберите файл конфигурации из вашей файловой системы
4. Нажмите **Открыть**, чтобы импортировать конфигурацию
5. Нажмите **Сохранить**, чтобы применить импортированные сочетания
6. Перезапустите приложение, чтобы изменения вступили в силу

## Лучшие практики для сочетаний клавиш

1. **Изучите распространенные сочетания**: Начните с самых часто используемых сочетаний
2. **Настройте под ваш рабочий процесс**: Настройте сочетания в соответствии с вашим предпочитаемым рабочим процессом
3. **Избегайте конфликтов**: Убедитесь, что новые сочетания не конфликтуют с существующими
4. **Документируйте ваши сочетания**: Ведите запись ваших пользовательских сочетаний
5. **Практикуйте регулярно**: Используйте сочетания часто, чтобы развить мышечную память

## Устранение проблем с сочетаниями клавиш

### Сочетания не работают
- **Проверьте фокус**: Убедитесь, что правильное представление имеет фокус
- **Проверьте конфликты**: Убедитесь, что другое приложение не перехватывает сочетание
- **Проверьте настройки**: Убедитесь, что сочетание включено в настройках
- **Перезапустите приложение**: Попробуйте перезапустить приложение
- **Сбросьте сочетания**: Если проблемы сохраняются, сбросьте сочетания к значениям по умолчанию

### Конфликты сочетаний
- **Определите конфликтующее приложение**: Определите, какое приложение вызывает конфликт
- **Измените сочетание**: Измените сочетание в настройках
- **Отключите конфликт**: Отключите конфликтующее сочетание в другом приложении

### Пользовательские сочетания не сохраняются
- **Проверьте разрешения**: Убедитесь, что у вас есть права на запись в файл настроек
- **Перезапустите приложение**: Перезапустите приложение, чтобы применить изменения
- **Сбросьте и переconfigure**: Сбросьте сочетания к значениям по умолчанию и переconfigure
