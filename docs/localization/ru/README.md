# Документация пользователя AVA AIGC Toolbox

> **Русская версия является официальной**

Добро пожаловать в полную документацию пользователя для AVA AIGC Toolbox! Эта документация проведет вас через все аспекты использования этого мощного кроссплатформенного приложения для управления изображениями, сгенерированными ИИ.

## Введение

AVA AIGC Toolbox — это функциональное приложение, предназначенное для управления, организации и манипулирования растущей коллекцией изображений, сгенерированных ИИ. Построенное на Avalonia и .NET, оно обеспечивает единый опыт на операционных системах Windows, macOS и Linux.

## Обзор документации

Документация организована так, чтобы помочь вам быстро начать работу, а затем изучать более сложные функции по мере знакомства с приложением.

## Системные требования

| Категория | Минимальные требования | Рекомендуется |
|----------|---------------------|-------------|
| **Операционная система** | Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+, Fedora 32+) | Windows 11, macOS 12+, последняя версия Linux |
| **.NET Runtime** | .NET 7.0 или новее | .NET 7.0 или новее |
| **Дисковое пространство** | 100 MB свободного места | 500 MB или больше |
| **Оперативная память** | 2 GB | 4 GB или больше |
| **Процессор** | Двухядерный процессор | Четырехядерный процессор или лучше |
| **Монитор** | Разрешение 1024x768 | 1920x1080 или выше |
