# AVA AIGC 工具箱 用户文档

> **备注：以英文版本为准**

欢迎使用 AVA AIGC 工具箱的全面用户文档！本文档将指导您使用这个功能强大的跨平台 AI 生成图像管理应用程序的各个方面。

## 介绍

AVA AIGC 工具箱是一个功能丰富的应用程序，旨在帮助您管理、组织和操作不断增长的 AI 生成图像集合。它使用 Avalonia 和 .NET 构建，在 Windows、macOS 和 Linux 操作系统上提供统一的体验。

## 系统要求

| 类别 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **操作系统** | Windows 10+，macOS 10.15+，Linux（Ubuntu 20.04+，Fedora 32+） | Windows 11，macOS 12+，最新 Linux 发行版 |
| **.NET 运行时** | .NET 7.0 或更高版本 | .NET 7.0 或更高版本 |
| **磁盘空间** | 100 MB 可用空间 | 500 MB 或更多 |
| **内存** | 2 GB | 4 GB 或更多 |
| **处理器** | 双核 CPU | 四核 CPU 或更好 |
| **显示器** | 1024x768 分辨率 | 1920x1080 或更高 |

## 支持的图像格式

- JPEG/JPG
- PNG
- WebP
- TIFF
- BMP
- GIF（仅静态）
