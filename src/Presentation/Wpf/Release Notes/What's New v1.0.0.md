# What's New in v1.0.0

Welcome to BerryAIGen.Toolkit v1.0.0! This is the initial release of our comprehensive image metadata indexer and viewer designed specifically for AI-generated images.

## Key Features

### Metadata Extraction
- Extracts detailed metadata (PNGInfo) from various AI image generation platforms
- Supports AUTOMATIC1111, InvokeAI, NovelAI, Stable Diffusion, EasyDiffusion, Fooocus family, and Stable Swarm

### Search and Filtering
- Advanced search capabilities across all extracted metadata
- SQLite Full-Text Search for fast text searching
- Dynamic query generation for complex search criteria

### Image Organization
- Ratings, tags, and albums for organizing images
- Support for batch operations on images

### Performance Optimizations
- Efficient thumbnail generation for quick preview
- Background processing for thumbnail generation
- Caching system for improved performance

### User Interface
- Intuitive WPF interface with MVVM architecture
- Multi-language support
- Responsive design with wide sidebar layout

## Technical Details

### Architecture
- Modular architecture with clear separation of concerns
- .NET 8.0 framework
- SQLite database for data storage
- GitHub integration for update checking

### Getting Started
1. Launch the application
2. Select folders containing your AI-generated images
3. Wait for the initial scan to complete
4. Start exploring and organizing your images
